Blur / Sharpen

Blur (clip, float amount)
Blur (clip, float amountH, float amountV)
Sharpen (clip, float amount)
Sharpen (clip, float amountH, float amountV)

Este é um filtro de núcleo 3x3 simples que borra. O maior argumento permissível para Blur é aproximadamente 1.58 que corresponde um píxel de borrar de até 3x3. Um valor de 1.0 o pega um núcleo de (1/4,1/2,1/4). Se você quer um grande raio de borrão Gaussian, recomendo encadear várias cópias de Blur(1.0). (Alguém se lembra do triângulo de Pascal?) .

Argumentos negativos para Blur de fato aguçam a imagem e na realidade Sharpen(n) é apenas um pseudônimo para Blur(-n). O menor argumento permissível para Blur é -1.0 e o maior para Sharpen é 1.0. 

Você pode usar 2 argumentos para fixar quantias Verticais e Horizontais independentes. Assim, você pode usar Blur(0,1) para filtrar só Verticalmente, por exemplo para misturar linhas entrelaçadas. Por padrão amountV=amountH.
