Tweak

Tweak (clip, float "hue", float "sat", float "bright", float "cont", bool "coring", bool "sse")

Esta função tem meios para ajustar a cor, saturação, brilho e contraste de um clipe de vídeo.

Hue: (-180.0 a +180.0, padrão 0.0) é usado para ajustar a tonalidade da imagem. Valores positivos trocam a imagem para vermelho. Valores negativos trocam para verde.

Sat: (0.0 a 10.0, padrão 1.0) é usado para ajustar a saturação da cor da imagem. Valores acima de 1.0 aumentam a saturação. Valores abaixo de 1.0 a reduzem. Use sat=0 para converter a cinza.

Bright: (-255.0 a 255.0, padrão 0.0) é usado para mudar o brilho da imagem. Valores positivos aumentam o brilho. Valores negativos diminuem o brilho.

Cont: (0.0 a 10.0, padrão 1.0) é usado para mudar o contraste da imagem. Valores acima de 1.0 aumentam o contraste. Valores abaixo de 1.0 diminuem o contraste.

coring = true/false (opcional; true por padrão, que reflete o comportamento em versões mais velhas) é incluído. Quando fixado em true o luma (Y) é cortado [16,235] e quando fixado em false, significa que luma permanece intacto. [Incluso na v2.53].

sse = true/false (opcional; false por padrão) re-habilita o código SSE se for preciso (talvez um AMD pode rodar mais rápido). [Incluso na v2.56].
