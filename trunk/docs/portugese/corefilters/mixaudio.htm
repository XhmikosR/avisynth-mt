MixAudio

MixAudio (clip1, clip2, float "clip1_factor", float "clip2_factor")

Mistura o áudio de dois clipes. Um volume para os dois clipes pode ser dado, mas é opcional.

O volume é dado como fator onde 0.0 é sem áudio do canal desejado e 1.0 é 100% do canal desejado. O padrão é 0.5/0.5 - se só um fator for determinado, o outro canal será 1.0-(factor). Se factor1 + factor2 é mais que 1.0, você arrisca recortar seu sinal.

Os dois clipes têm que ter áudio nas mesmas taxas de amostra (use ResampleAudio se este for o problema). Seus clipes precisam ter o mesmo número de canais (estéreo/mono) - use ConvertToMono  ou MergeChannels se este for o problema.

  

# Misturar duas fontes, com uma ligeiramente mais baixa que a outra.
Soundtrack = WavSource("c:\soundtrack.wav")
Speak = WavSource("c:\speak.wav")
return MixAudio(Soundtrack, Speak, 0.75, 0.25)    # O Expert pode notar que o último 0.25 é realmente redundante aqui.
