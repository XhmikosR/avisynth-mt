FixLuminance

FixLuminance (clip, int intercept, int slope)

Meu videocassete tem o mau hábito de fazer o topo de cada quadro mais luminoso que o fundo. O propósito deste filtro é escurecer o topo da imagem progressivamente para compensar isto. Se você notou um problema semelhante com seu videocassete, mande um e-mail e eu terminarei esta explicação de como o filtro trabalha. Caso contrário, você finja que não existe.

Este filtro só trabalha com entradas YUY2.
