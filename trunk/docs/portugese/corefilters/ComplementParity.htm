ComplementParity(clip clip) 

Se o clipe de entrada for baseado em campo, ComplementParity muda os campos de topo para os campos inferiores e vice-versa. Se o clipe de entrada for baseado em quadro, muda o campo dominante de cada quadro (fundo dominante para topo dominante e vice-versa).
