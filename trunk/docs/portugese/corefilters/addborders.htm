AddBorders

AddBorders (clip, int left, int top, int right, int bottom, int "color")

AddBorders Põe bordas pretas ao redor da imagem, com as larguras especificadas (em píxeis).

Veja colorspace conversion filters para forçar o uso de formatos de cores diferentes.

O parâmetro de cor é opcional (incluído na v2.07), default=0 <black>, e é especificado como um valor RGB embora o formato de clipe é baseado em RGB. Os pré-ajustes podem ser achados no arquivo colors_rgb.avsi que deve estar presente em sua pasta de plugin.

Esteja atento que muitos algoritmos de compressão com perda não tratam bem as bordas de cores sólidas, a menos que a largura da borda seja um múltiplo do tamanho do bloco (16 píxeis para MPEG).

Você pode usar este filtro combinado com Crop para trocar uma imagem em volta sem mudar o tamanho de quadro. Por exemplo: 

# Muda uma 352x240 imagem 2 píxeis à direita
Crop(0,0,350,240).AddBorders(2,0,0,0)
# Coloca bordas de letterbox que são [16,16,16] pretas (para CCIR-601) fazendo uma imagem 720x308 em 720x480
AddBorders(0, 86, 0, 86, $000000)
