SelectEven / SelectOdd

SelectEven (clip)
SelectOdd (clip)

SelectEven faz uma produção de vídeo usando só os quadros de entrada numerados. SelectOdd é sua contraparte diferente.

Desde que os quadros são numerados a partir de zero, SelectEven seleciona o primeiro, terceiro, quinto,... quadros pelas convenções de contagem humanas.
