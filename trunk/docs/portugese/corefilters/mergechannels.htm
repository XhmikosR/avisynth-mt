MergeChannels

MergeChannels (clip1 , clip2 [, clip3, ...])

A partir da v2.5 MergeChannels substitui MonoToStereo, e pode ser usado para fundir os canais de áudio de dois ou mais clipes.
Não confunda isto com mistura de canais (MixAudio e ConvertToMono fazem isso) - o som de cada canal é intocado, os canais só são postos no novo clipe.
Antes de fundir o áudio é convertido ao tipo de amostra do clip1.

  

# Exemplo, converte áudio "wav sem compressão" para um sinal estéreo de 44.1kHz:
video = AviSource("c:\divx_wav.avi") 
audio = WavSource("c:\divx_wav.avi") 
l_ch = GetChannel(audio, 1) 
r_ch = GetChannel(audio, 2) 
stereo = MergeChannels(l_ch, r_ch).ResampleAudio(44100) 
return AudioDub(video, stereo) 

# Isto é semelhante para: 
video = AviSource("c:\divx_wav.avi") 
audio = WavSource("c:\divx_wav.avi") 
stereo = GetChannel(audio, 1, 2).ResampleAudio(44100) 
return AudioDub(video, stereo)
