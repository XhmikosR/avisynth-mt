AlignedSplice / UnalignedSplice

AlignedSplice (clip1, clip2 [, ...])
UnAlignedSplice (clip1, clip2 [, ...])

AlignedSplice e UnalignedSplice une une dois ou mais clipes de vídeo. A diferença entre os filtros é o modo como tratam a trilha sonora. UnalignedSplice concatena o som sem levar em conta a sincronização com o vídeo.

AlignedSplice corta a primeira trilha sonora ou insere silêncio se necessário para assegurar que a segunda trilha sonora permaneça sincronizada com o vídeo.

Você deve usar UnalignedSplice quando as trilhas sonoras que são unidas eram originalmente contíguas - por exemplo, quando você está unindo arquivos capturados com AVI_IO. Erros de curtos momentos podem produzir defeitos no som se você usar AlignedSplice nestas situações.

Os escritos de Avisynth têm operadores + e ++ como sinônimos para UnalignedSplice e AlignedSplice respectivamente.

Veja aqui as propriedades resultantes do clipe. 

# Unir os arquivos de captura segmentada para produzir um único clipe
UnalignedSplice(AVISource("cap1.avi"),AVISource("cap2.avi"),AVISource("cap3.avi"))
# ou: AVISource("cap1.avi") + AVISource("cap2.avi") + AVISource("cap3.avi")

# Extrair três cenas de um clipe e uní-los em uma nova ordem
AVISource("video.avi")
edited_video = Trim(2000,2500) ++ Trim(3000,3500) ++ Trim(1000,1500)
