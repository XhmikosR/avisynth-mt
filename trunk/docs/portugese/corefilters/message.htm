MessageClip

MessageClip (string message, int "width", int "height", bool "shrink", int "text_color", int "halo_color", int "bg_color")

MessageClip produz um clipe que contém uma mensagem de texto; usado internamente para informar erro. Fonte Arial é usada, tamanho entre 24 e 9 pontos escolhidos para ajustar no clipe, se possível, na largura e altura.
Se shrink é true, a resolução do clipe é então reduzida para ajustar o texto, se necessário.
