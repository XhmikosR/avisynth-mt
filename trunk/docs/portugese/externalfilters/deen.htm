Descrição

deen é um conjunto de "denoisers" (removedores de ruído) otimisados em linguagem assembly.

uso

deen (clip, string "name", int "mode", int "thrY", int "thrUV", float "ti")

name :
você escolhe o método de "denoising" (remoção de ruído) aqui (c3d, a3d, a2d, m2d).
padrão: "c3d"

mode :
com "c3d", é a matriz de peso.
- 0 < 121 > matriz (padrão)
- 1 < 111 > matriz (mais suave)
com outros métodos, é o raio espacial.
padrão: 0

thrY, thrUV :
percentuais luma (Y) e croma (UV).
valores mais altos misturam mais.
padrão: 10,12

ti :
influência temporal.

maximal significa diferença temporal local

permitido habilitar filtrado 3d.

padrão é 3

métodos

vários métodos estão implementados:

- este lançamento é uma prévia do modo "c3d".

é completamente funcional e os resultados estão muito

muito pertos do filtro Convolution3D de vlad.

 

- você pode testar o modo "a3d" e "a2d".

é meu próprio implementador do denoising 2d/3d.

está em fase beta; você pode tentar deen("a3d",1,10,12)

como uma alternativa para c3d com matriz <111> ou

deen("a2d",2,10,12) para filtro espacial 5x5.

 

- m2d " está em fase alfa.

mas você pode dar uma olhada em deen("m2d",1,10,12)
