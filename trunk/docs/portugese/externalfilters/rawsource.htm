O que ele faz

Este filtro carrega dados de vídeos raw (crus).

Uso

RawSource (string filename, int width, int height, string pixel_type)

filename: o arquivo raw (cru) por exemplo um YUV.
width, height: você tem que especificar as dimensões da imagem. A width (largura) máxima é 2880.
pixel_type: o tipo dos dados raw. Um mapeamento apropriado para os dados internos de AviSynth é feito. Tipos suportados: RGB, RGBA, BGR, BGRA, YUYV, UYUV, YV12, I420,.

O framerate é fixado a 25 fps, você pode mudar isto com AssumeFPS, se precisar (por exemplo, em material NTSC).

Exemplo:

LoadPlugin("c:\myprojects\rawsource\release\rawsource.dll")

# RawSource("d:\avitest\src6_ref__625.yuv",720,576,"BGRA")
RawSource("d:\blue_sky.yuv.GetRight",1920,1080,"YV12")
bilinearresize(352, 288)

# salve como arquivo bmp:
ConvertToRGB32()
imagewriter("d:\", 0, 2, "ebmp")

Aqui está uma ligação contendo avi2yuv  que pode ser usado para fazer arquivos crus yuv. Você precisa alimentar isto com arquivos sem compressão RGB, caso contrário não trabalha.
