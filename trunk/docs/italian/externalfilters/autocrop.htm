AutoCrop

Sommario

author: Glenn Bussell
version: 1.1
download: http://www.videofringe.com/autocrop
category: Broadcast Video Plugins
requirements: YV12 & YUY2 Colorspace

Descrizione

AutoCrop è un filtro di AVISynth che automaticamente croppa i bordi neri di un clip. Opera sia in modo preview (dove sovrappone le informazioni per il cropping consigliato) , che in modo cropping col quale croppa realmente il clip. Può inoltre garantire che larghezza ed altezza siano multipli di un valore specificato in modo tale che il clip croppato possa essere passato senza problemi al compressore video.

Uso

AutoCrop accetta 10 parametri tutti opzionali sicché basta AutoCrop() per abituarsi a come lavora il filtro. I parametri possono essere specificati come posizionali o per nome. Suggerisco di specificarli per nome visto che sono tanti!

Orine dei parametri.

AutoCrop ("mode", "wMultOf", "hMultOf", "leftAdd", "rightAdd", "topAdd", "bottomAdd", "threshold", "samples", "aspect")

Sintassi

mode - Intero - default 1

0 - Crop - Croppa l'immagine
1 - Preview - Le informazioni di cropping consigliate vengono sovrapposte al clip, incluso un comando di crop che può essere usato al posto di AutoCrop.
2 - Log - Crea un logs dei parametri di cropping nel file "AutoCrop.log" nella directory currente.

wMultOf - Intero- default 4

Garantisce che la larghezza sia un multiplo di un valore specificato in modo tale che il clip croppato possa essere passato senza problemi al compressore video.

hMultOf - Intero - default 2

Idem per l'altezza.

leftAdd - Intero - default 0

Forza autocrop a croppare ulteriori leftAdd pixels dopo aver completato la auto individuazione. Si può usare questa opzione congiuntamente a wMultOf. Utile per croppare le linee di pessima qualità a lato delle catture VHS.

rightAdd - Intero - default 0

Come sopra riferito al lato destro.

topAdd - Intero - default 0

Forza autocrop a croppare ulteriori topAdd pixels dopo aver completato la auto individuazione. Si può usare questa opzione congiuntamente a hMultOf. Utile per croppare le linee di pessima qualità in cima alle catture VHS.

bottomAdd - Intero - default 0

Come sopra riferito al lato inferiore.

threshold - Intero (0-255) - default 40

Threshold è la luminanza media che una riga deve avere prima di essere considerata non nera.

samples - Intero - default 10

Numero di frames da esaminare per determinare le informazioni di cropping. Questo influenza direttamente il tempo di inizio lavoro del filtro. In generale 10 sembra essere tanto, se questo ritardo vi infastidisce, abbassatelo.

aspect - Float - default 0 - L'aspetto è ignorato.

-1 - Mantiene l'aspect ratio del clip
0 - L'aspetto è ignorato.
>0 - L'aspect ratio sarà impostato a questo valore. Ad esempio per una aspect ratio 4:3 si deve passare
4.0/3.0 o 1.33333333333. Notare che 4/3 non funziona e che .0 è richiesto per dire ad AviSynth che il valore è float.

Uso consigliato

Suggerisco vivamente di usare il modo preview prima di sottomettere un crop particulare . Vale la pena di notare che copiare i settaggi suggeriti nel comando di crop standard di AviSynth è più veloce che usare AutoCrop con mode = 0.
