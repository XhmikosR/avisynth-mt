RGBAdjust

RGBAdjust (clip, float red, float green, float blue, float alpha)

Questo filtro moltiplica ogni colore per un valore dato, tagliando il risultato a 255 se maggiore. Si noti che RGBAdjust(1, 1, 1, 1) lascia la clip inalterata.

red (0.0 - 255.0): Questa opzione determina il fattore di scala per il Rosso. Ad esempio un fattore x3 moltiplica il colore rosso di ogni pixel per 3. Allo stesso modo per il Verde ed il Blu.
alpha (0.0 - 255.0) specifica l'alpha channel, che identifica l'informazione di trasparenza di ogni pixel. Il valore 0 di alpha rappresenta la trasparenza completa, mentre il valore 255 rappresenta un pixel completamente opaco.

Si tenga presente che i valori NON sono moltiplicati per adattarsi a quanto fatto su un colore (per la qual cosa si deve usare i "levels") sicchè facendo : 

RGBAdjust(2, 1, 1, 1)

si otterranno frames tutti rossi. Se questo è il vostro obiettivo,OK, altrimenti se volete avere PIU' rosso dovete scrivere

Levels(0, 1, 255, 0, 128).RGBAdjust(2, 1, 1, 1)

TQuesto dimezza tutti i livelli (compresa la luminanza) e poi raddoppia il rosso.
