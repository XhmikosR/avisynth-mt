MessageClip

MessageClip (string message, int "width", int "height", bool "shrink", int "text_color", int "halo_color", int "bg_color")

MessageClip produce un clip contenente un testo; usato internamente per error reporting.
E' usato un Arial font , size tra 24 points e 9 points scelta per adattarsi , se possibile, nel clip di dimensioni width e height .
Se shrink è true, la risoluzione del clip viene ridotta , se necessario, per adattarsi al testo.
