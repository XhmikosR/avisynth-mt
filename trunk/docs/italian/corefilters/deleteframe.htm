DeleteFrame

DeleteFrame (clip, int frame)

DeleteFrame cancella un singolo frame, passato come argomento. La traccia audio non viene modificata, sicchè usandolo troppe volte, si ha una evidente desincronizzazione.

Se si vuole tagliare una porzione di clip (da a a b, diciamo), assieme alla corrispondente porzione di soundtrack, lo si può fare con Trim :

Trim(0,a-1) ++ Trim(b+1,0)
