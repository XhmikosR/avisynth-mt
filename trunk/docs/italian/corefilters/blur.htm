Blur / Sharpen

Blur (clip, float amount)
Blur (clip, float amountH, float amountV)
Sharpen (clip, float amount)
Sharpen (clip, float amountH, float amountV)

Questo è un semplice filtro blurring 3x3. Il valore più grande ammesso per l' argumento del Blur è circa 1.58, che corresponde ad un blur di 3x3-pixel. Il valore 1.0 produce un kernel (1/4,1/2,1/4). Se si desidera un blur Gaussiano a più ampio raggio, raccomando di contatenare assieme diverse copie di Blur(1.0) . ( Nessuno ricorda il triangolo di Pascal ?) 

Valori negativi degli argumenti del Blur intensificano l' immagine, ed in effetti Sharpen(n) è un alias di Blur(-n). Il valore più basso utilizzabile come argomento di Blur è -1.0 ed il più grande per Sharpen è 1.0. 

Si possono usare 2 argumenti per settare valori indipendenti Verticali e Orizzontali. In questo modo, si può usare Blur(0,1) per filtrare solo Verticalmente, per esempio per fondere assieme linee interallacciate. Di default amountV=amountH.
