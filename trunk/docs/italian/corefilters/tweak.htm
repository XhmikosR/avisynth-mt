Tweak

Tweak (clip, float "hue", float "sat", float "bright", float "cont", bool "coring")

Questa funzione fornisce gli strumenti per modificare hue, saturation, brightness, e contrast di un video clip.

Hue: (-180.0 to +180.0, default 0.0) è usato per aggiustare il colore dell'immagine. Valori positivi spostano l'immagine verso il roso. Quelli negativi verso il verde.

Sat: (0.0 to 10.0, default 1.0) è usato per modificare la saturazione dell'immagine. Valori superiori a 1.0 aumentano la saturazione. Valori al di sotto di 1.0 la riducono. Si usa sat=0 per passare alla Scala dei grigi (Greyscale).

Bright: (-255.0 to 255.0, default 0.0) è usato per cambiare la luminosità dell'immagine. Valori positivi la aumentano , quelli negativi la diminuiscono.

Cont: (0.0 to 10.0, default 1.0) è usato per cambiare il contrasto dell'immagine. Valori superiori a 1.0 lo aumentano, valori inferiori a 1.0 lo diminuiscono.

coring = true/false E' stato aggiunto (optionale; true di default, che rispecchia il comportamento delle vecchie versioni). Impostato a true significa che (Y) è limitato a [16,235], quando a false il luma rimane invariato. [Aggiunto in v2.53.]
