SelectRangeEvery

SelectRangeEvery (clip, int "every", int "length", int "offset", bool "audio")

Questo filtro è disponibile a partire dalla v2.5. Esso seleziona un length numero di frames every (ogni) n frames, a partire da offset. I valori di default sono: Select 50 frames ogni 1500 frames, iniziando dal frame frame 0. (every = 1500, length = 50, offset = 0).

Dalla v2.55, SelectRangeEvery elabora anche l'audio. Per mantenere l'audio originale , usare audio = false.

Examples: 

# Selects the frames 0 to 13, 280 to 293, 560 to 573, etc.
SelectRangeEvery(clip, 280, 14)

# Selects the frames 2 to 15, 282 to 295, 562 to 575, etc.
SelectRangeEvery(clip, 280, 14, 2)
