Tone

Tone (float "length", float "frequency", int "samplerate", int "channels", string "type", float "level")

This will generate sound at a given frequency for a given length of time in seconds. Type can be "Silence", "Sine" (default), "Noise", "Square", "Triangle" or "Sawtooth".

Defaults are Tone(10.0, 440, 48000, 2, "sine", 1.0).
