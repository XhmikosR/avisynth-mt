Greyscale

Greyscale (clip [, string "matrix"])

Convertit en niveaux de gris (sans changer le format de couleur).

La conversion est effectuée en utilisant les coefficients du standard Rec.601. Dans v2.56, une option matrix a été ajoutée. En indiquant matrix="Rec.709", le clip est convertit en niveaux de gris en utilisant les coefficients du standard Rec.709. Voir le paragraphe Conversions de couleurs pour une explication sur ces coefficients. Quand vous indiquez matrix="Average" (moyenne) (seulement pour les clips RGB), la luminance est calculée telle que (R+G+B)/3.
