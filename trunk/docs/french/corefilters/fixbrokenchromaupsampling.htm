FixBrokenChromaUpsampling

FixBrokenChromaUpsampling (clip)

Le codec Canopus gratuit dans sa version 1.00 échantillonne mal les canaux(bien que les verions plus récentes non grtuites semblent corriger ce problème). Chroma est dupliqué depuis un autre champ (en anglais), produisant ainsi la erreur d'échantillonnage de Chroma.

Le filtre FixBrokenChromaUpsampling corrige cette erreur. Vous devez le placer après AviSource si vous utilisez le codec DV de Canopus. Les ancienne versions DirectShow du codec MS DV peut également avoir ce problème (la version fourni avec DirectX7 (mais je dois vérifier cela), celle fourni avec DirectX8/9 fonctionne correctement).

Le codec Canopus inverse le chroma des deux lignes du milieu d'un groupe de quatre lignes:

fimage_correcte image_Canopus
ligne 1 ligne 1
ligne 2 ligne 3
ligne 3 ligne 2
ligne 4 ligne 4
ligne 5 ligne 5
ligne 6 ligne 7
ligne 7 ligne 6
ligne 8 ligne 8

FixBrokenChromaUpsampling corrige cela en échangeant le paramètre chroma aux valeurs correctes.
