Interleave

Interleave (clip1, clip2 [, ...])

Interleave entremêle des images de plusieurs clips sur une base image par image. Par exemple, si vous donnez trois arguments à ce filtre, les trois premières image du clip de sortie seront  les premières images des trois clip sources, les trois suivante seront les deuxièmes images des trois clips sources, et ainsi de suite.

Voir également ici pour plus d'infos sur les propriétés du clip résultant.
