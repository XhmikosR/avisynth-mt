Import

Import (string filename)

Import évalue le contenu d'un autre script AviSynth.

Cela peut être utile pour enregistrer différentes fonctions, des variables globales, ou des flux pré-construits. Rigoureusement parlant, c'est une fonction de script plutôt qu'un filtre.
