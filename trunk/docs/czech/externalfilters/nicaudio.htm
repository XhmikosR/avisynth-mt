Popis

AviSynth audio Pluginy pro MPEG Audio/AC3/DTS/LPCM a další nekomprimované formáty.

Syntaxe

AC3: NicAC3Source ("FileName.ac3", int "Channels", int "DRC")
DTS: NicDTSSource ("FileName.dts", int "Channels", int "DRC")
MPA: NicMPG123Source ("FileName.mpa", bool "Normalize")
LPCM: NicLPCMSource ("FileName.lpcm", int "SampleRate", int "SampleBits", int "Channels")
RAW: RaWavSource ("FileName.raw", int "SampleRate", int "SampleBits", int "Channels")

PARAMETRY

Channels Maximální poèet kanálù pro výstup (Downmix). Je to jen volitelný parametr. Nutný však pro lpcm a raw soubory. Max 8 kanálù.

DRC Použije algoritmus Dynamic Range Compression. 0 znamená nic (výchozí), 1 znamená Normal

Normalize Použije maximální zesílení (gain) bez oøíznutí signálu. Výchozí false.

SampleRate Nutný pro lpcm a raw soubory. V RaWavSource hodnota < 9 znamená hlavièku s platnými hodnotami pro SampleRate/SampleBits/Channels a nyní znamená:

    IgnoreLength
  • 1 vynutí ignorovat ètení dat velikosti také v 64 bitových formátech.
  • 2 ignoruje ètení dat velikosti 32 bitù v hlavièce pokud > 2GB. (výchozí)
  • 4 ignoruje ètení dat velikosti 32 bitù v hlavièce pokud > 4GB.

Podporované soubory (všechny mohou být > 4GB):

ac3 (Dopracovat: podporu také eac3 souborù nebo Dolby Digital Plus)
dts také jsou podporovány dtswav
mpa mpeg soubory: mp1, mp2 a mp3
lpcm z DVD Audio, z BluRay se - SampleBits
raw a nekomprimované formáty: WAV, WAVE_FORMAT_EXTENSIBLE, W64, BWF, RF64, AU, AIFF a CAF
