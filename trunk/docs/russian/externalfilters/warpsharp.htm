WarpSharp

Общая информация

Автор: аноним (японец)
Версия: warpsharp_2003_1103.cab
Загрузка: http://www.geocities.co.jp/SiliconValley-PaloAlto/2382/ (английское введение)
Категория: Плагины для повышения / смягчения резкости
Требования: YUY2 & YV12 цветовой формат

Вам также необходимо скопировать файлы msvcp71.dll и msvcr71.dll в вашу системную директорию.

Описание

Синтаксис следующий:

WarpSharp (clip, int depth = 128, int blur = 3, int bump = 128, float cubic = -0.6)

Описание параметров:

depth
Глубина деформации.

blur
Число операций размывания (блюра). Обработка замедляется, если эта величина увеличивается. Если эта величина не определена, устанавливается в минимум, 1.

bump
Порог регистрации неровности (unevenness). Эффект повышения резкости усиливается, если порог увеличивается.

cubic
Коэффициент величины деформации (кубическая интерполяция). Нет необходимости менять эту настройку.
