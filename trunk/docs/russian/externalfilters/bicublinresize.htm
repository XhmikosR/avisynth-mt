Описание (Description)

Это набор фильтров для изменения разрешения отсчетов / изменения размеров (resamplers/resizers) для Avisynth 2.5. Все эти размеро-изменяющие фильтры нефильтрующие (unfiltered), то есть они чертовски быстрые, хоть и далеки от совершенства, т.к. в них используется малое количество точек выборки. В любом случае, для повседневных задач они могут прекрасно сработать ^^

Кстати (BTW), очень важно протестировать их и на СВОЙ СОБСТВЕННЫЙ взгляд принимать решение о их пригодности. Причина проста: хоть это не полноценные фильтрующие интерполяторы, результат выглядит вполне прилично, поверьте.

Синтаксис (Syntax)

FastBilinearResize (h, v)
FastBicubicResize (h, v, "b", "c")
BicublinResize (h, v, "b", "c')

FastBilinear подобен SimpleResize от trbarry.
FastBicubic - нефильтрующий размеро-изменяющий бикубический фильтр.
Bicublin  - использует бикубическое преобразование для яркости Y и билинейное для цветности UV.

h, v = горизонтальное и вертикальное разрешение.
b, c = бикубические коэффициенты (bicubic coefficients). По умолчанию: b = 1/3, c = 1/3
