aWarpSharp

Общая информация

Автор: Marc FD
Версия: beta 1
Загрузка: http://www.avisynth.org/warpenterprises/
Категория: Плагины для повышения / смягчения резкости
Требования:  
  • YV12 цветовой формат
  • ISSE поддержка

Описание

синтаксис:

aWarpSharp (float "depth", int "blurlevel", float "thresh", int "cm")

depth & blurlevel (глубина и уровень размывания):
Это настройки, с которыми вы могли быть знакомы по фильтру WarpSharp для VirtualDub. Они являются единственными, которые вы должны подобрать, чтобы достичь какого либо эффекта. Blurlevel нагружает процессор, но дает большой скачок к повышению резкости деформацией.
по умолчанию : 16.0 & 2

thresh : (порог)
Величина с плавающей точкой. 1.00 означает 100% (max). Это есть настройка насыщения отображения сжатия. Величина по умолчанию рекомендуется для максимального качества. Если вы подгоняете эту настройку, держите в мысли, что она может усиливать неравенство деформации между краями.
по умолчанию : 0.5 (50%)

cm : режим цветности (chroma mode)
cm = 0 отменяет фильтрацию цветности.
cm = 1 разрешает деформацию цветности с отображением сжатия яркости (рекомендуется).
cm = 2 разрешает независимую деформацию цветности & отображение сжатия (bump map).
по умолчанию : 2

Продвинутые настройки недокументированы. Он не нужны в любом случае. (Я не использую их, и я думаю, лучше если никто не будет использовать их ^^)

О качестве & скорости

aWarpSharp реализует высококачественное оригинальное повышение резкости деформацией (warpsharping). "Оригинальное" означает, что использованный алгоритм отличается по некоторым пунктам от других подобных фильтров. Код полностью оптимизирован для iSSE с высокой точностью, смотрите сами ^^. Вам необходим примерно 550 Mhz процессор чтобы обработать изображение 640x480 при 25 кадрах в секунду. Это худший случай, я достигал около 40 fps при полном разрешении PAL MPEG-2 декодировании, деинтерлейсе & повышении резкости используя MPEG2Dec("dvd.d2v").aDeInt().aWarpSharp() на 1.4 Ghz процессоре.

aWarpSharp очень оптимизирован для моего процессора (Athlon XP), но все современные процессоры будут выполнять этот фильтр с максимальной допустимой скоростью вашего железа ^_^.

Я надеюсь вам понравится мой последний фильтр. Это есть одни из лучших кусков кода, которые я когда либо писал, и было очечь приятно программировать.
