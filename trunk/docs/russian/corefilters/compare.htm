Compare

Compare (clip_filtered, clip_original, string "channels", string "logfile", bool "show_graph")

Compare выводит отфильтрованный клип (первый параметр) с текстом (среднее абсолютное различие, среднее различие и PSNR) и с PSNR графиком Марка (если show_graph имеет значение true, по умолчанию). Начиная с v2.53 'Overal PSNR' также показывается на экране, но пока еще не записывается в лог-файл. Строка "channels" это комбинация из R,G,B [,A] или Y,U,V, в зависимости от исходного клипа. Пустая строка (по умолчанию) означает или "YUV" или "RGB". Если "logfile" не пропущено, результат будет записан в файл с этим "именем" и не будет нарисован на клипе. Это гораздо быстрее, если вам нужно сравнить много кадров.

  

# Показывает различия на экране
Compare(clip1, clip2)
# чтобы создать лог файл:
Compare(clip1, clip2, "", "compare.log")
# будет сравнен только сигнал канала цветности в YUY2 клипе.
Compare(clip1, clip2, "UV")
