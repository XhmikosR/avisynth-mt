FreezeFrame

FreezeFrame (clip, int first_frame, int last_frame, int source_frame)

Фильтр FreezeFrame заменяет все кадры между first_frame и last_frame копией source_frame. Звуковая дорожка не модифицируется. Это полезно для покрытия проблемных кадров в видео, если имеется похожий беспроблемный кадр.
