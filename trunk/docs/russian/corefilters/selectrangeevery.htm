SelectRangeEvery

SelectRangeEvery (clip, int "every", int "length", int "offset", bool "audio")

Этот фильтр доступен начиная с v2.5. Фильтр выбирает число кадров length из каждых every кадров, начиная с кадра offset.
 Значения по умолчанию: Выбрать 50 кадров из каждых 1500 кадров, начиная первую выборку с кадра 0. (every = 1500, length = 50, offset = 0).

Начиная с v2.55, SelectRangeEvery также обрабатывает аудио. Чтобы сохранить оригинальное аудио, используйте audio = false.

Примеры: 

# Выбирает кадры с 0 по 13, с 280 по 293, с 560 по 573, и т.д.
SelectRangeEvery(clip, 280, 14)

# Выбирает кадры с 2 по 15, с 282 по 295, с 562 по 575, и т.д.
SelectRangeEvery(clip, 280, 14, 2)
