Greyscale

Greyscale (clip [, string "matrix"])

Конвертирует в серое (без изменения цветового формата).

Преобразование производится с использованием коэффициентов Rec.601. В v2.56 добавлена опция matrix. При установке matrix="Rec.709", клип преобразуется в серое с использованием коэффициентов Rec.709. См ColorConversions для объяснения этих коэффициентов. При установке matrix="Average" (только для RGB клипов) сигнал яркости рассчитывается как (R+G+B)/3.
