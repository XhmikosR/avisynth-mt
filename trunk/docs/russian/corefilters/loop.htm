Loop

Loop (clip, int "times", int "start", int "end")

Повторяет в цикле сегмент от кадра start до кадра end заданное в аргументе times число раз.
times (по умолчанию -1) это число повторений цикла.
start (по умолчанию 0) это кадр клипа, где начинается цикл.
end (по умолчанию framecount(clip)) это  кадр клипа, где цикл заканчивается.

  

# Повторяет в цикле кадры с 100 до 110 текущего клипа 10 раз
Loop(10,100,110)


Loop() # делает клип повторяющимся (почти) бесконечно


Loop(10) # делает клип повторяющимся 10 раз


Loop(10,20,29) # 10 раз повторяет кадры с 20 до 29 прежде чем продолжить
# реальная длительность клипа увеличится на 90 кадров


Loop(0,20,29) # удалить кадры с 20 по 29
# реальная длительность клипа уменьшится на 10 кадров


Loop(-1,20,29) # кадры с 20 по 29 повторяются (почти) бесконечное число раз
