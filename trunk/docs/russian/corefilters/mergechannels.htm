MergeChannels

MergeChannels (clip1 , clip2 [, clip3, ...])

Начиная с версии 2.5 функция MergeChannels заменила MonoToStereo, и может быть использована для слияния аудио-каналов из одного и более клипов.
Не путайте ее со смешиванием звука в каналах (что делают MixAudio и ConvertToMono) - звук каждого канала остается нетронутым, каналы просто добавляются в новый клип.
Перед слиянием звук конвертируется к типу первого клипа clip1.

 # Например, преобразовать "несжатый wav" звук в 44.1 кГц стерео сигнал: 

video = AviSource("c:\divx_wav.avi") 
audio = WavSource("c:\divx_wav.avi") 
l_ch = GetChannel(audio, 1) 
r_ch = GetChannel(audio, 2) 
stereo = MergeChannels(l_ch, r_ch).ResampleAudio(44100) 
return AudioDub(video, stereo) 

# что аналогично: 
video = AviSource("c:\divx_wav.avi") 
audio = WavSource("c:\divx_wav.avi") 
stereo = GetChannel(audio, 1, 2).ResampleAudio(44100) 
return AudioDub(video, stereo)
