FreezeFrame

FreezeFrame (clip, int first_frame, int last_frame, int source_frame)

FreezeFrame フィルタは、first_frame と last_frame の間のすべてのフレームを source_frame のコピーで置換します。音声トラックは修正されません。グリッチ*1のないよく似たフレームが利用可能な場合に、ビデオ内のグリッチを隠すのに役立ちます。
