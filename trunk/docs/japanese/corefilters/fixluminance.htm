FixLuminance

FixLuminance (clip, int intercept, int slope)

私のビデオデッキは、各フレームの上端を下端よりも明るくするという迷惑な癖を持ってます。このフィルタの目的は、これを相殺するために画像の上端を徐々に暗くすることです。もしあなたのビデオデッキに関して同様の問題に気づいているなら、私にメールしてください。そうすれば、私はこのフィルタがどのように動作するかの説明を完成させるでしょう。さもなければ、あなたは、このフィルタが存在しないふりをすることができます。

このフィルタは、YUY2 入力に対してのみ動作します。
