AviSynth のシンタックス - 内部関数

内蔵フィルタに加えて、AviSynth には他にかなりたくさんの（非クリップ型の）内部関数があります。これらの関数の入力または/および出力はクリップではなく、スクリプトの中で使用可能な別の変数です。これらの関数は、おおよそ次のように分類されます:

  • 数値関数
数値変数に関する一般的な数学演算を提供します。
  • 文字列関数
文字列変数に関する一般的な操作を提供します。
  • ブール関数
テストする条件があてはまるなら true を、さもなければ false をそれぞれ返します。
  • 変換関数
異なる型の間で変換を行います。
  • 制御関数
（スクリプトの読み込み、引数のチェック、グローバル設定の調節、といった）制御の流れを容易にします。
  • バージョン関数
AviSynth のバージョン情報を提供します。
  • ランタイム関数
フレームごとに評価される内部関数です。フレームに関する情報を返すためにランタイムフィルタ（ConditionalFilter、ScriptClip、FrameEvaluate）に渡されるスクリプトの内部で使用することができます。
