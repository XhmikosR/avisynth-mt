SuperEQ

SuperEQ (clip, string filename)

Shibatch Super Equalizer は、Winamp*1 用のグラフィックイコライザ*2兼パラメトリックイコライザ*3プラグインです*4。このプラグインは、FFT アルゴリズム*5を使った 16383 階調の FIR フィルタ*6を使用しています。その平均化は、非常に正確です。平均化の設定は、チャンネルごとに独立して行うことが可能です。SuperEQ は、元々、Naoki Shibata 氏によって書かれたものです。

SuperEQ は、foobar2000*7 のイコライザ設定ファイルを必要とします。イコライザは foobar の DSPManager にあり、設定はそこから調節され、保存されます。

いくつかのプリセット設定は、こちらからダウンロードすることができます。

このプラグインは、128 KB （16383*2*sizeof(float) = 128 KB）以上のキャッシュを持つ CPU 向けに最適化されています。このプラグインは、K6 シリーズ*8の CPU では効率的に機能しません（Athlon を買いましょう！！）。

パラメータ:

filename 適用する foobar2000 のイコライザプリセットファイル。

使用例:

上記のイコライザプリセットの「Loudless」フィルタを適用する:

SuperEq("C:\Equalizer Presets\Loudness.feq")
