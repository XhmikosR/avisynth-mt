AlignedSplice / UnalignedSplice

AlignedSplice (clip1, clip2 [, ...])
UnAlignedSplice (clip1, clip2 [, ...])

AlignedSplice と UnalignedSplice は、2 本以上のビデオクリップの端と端を結合します。2 つのフィルタの違いは、音声トラックの取り扱い方にあります。UnalignedSplice は、ビデオとの同期を考慮せず、単純に音声トラックを結合します。AlignedSplice は、2 本目の音声トラックと映像との同期の保持を確保するために、必要に応じて 1 本目の音声トラックをカットするか、または無音を挿入します。

たとえば AVI_IO*1でキャプチャされたファイルを結合するときのように、結合される音声トラックが元々連続しているときは UnalignedSplice を使用するべきです。このような状況で AlignedSplice を使うと、わずかなタイミングの誤差が音声内にグリッチノイズ*2を引き起こすかもしれません。

AviSynth のスクリプティング言語は、UnalignedSplice と AlignedSplice の同義語として、それぞれ + 演算子と ++ 演算子を提供します。

また、生成されるクリップのプロパティに関しては、こちらも参照してください。

# セグメント化されたキャプチャファイルを結合して、1 本のクリップを生成する
UnalignedSplice(AVISource("cap1.avi"),AVISource("cap2.avi"),AVISource("cap3.avi"))
# もしくは: AVISource("cap1.avi") + AVISource("cap2.avi") + AVISource("cap3.avi")

# クリップから 3 つのシーンを抽出して、それらを新しい順番で結合する
AVISource("video.avi")
edited_video = Trim(2000,2500) ++ Trim(3000,3500) ++ Trim(1000,1500)
