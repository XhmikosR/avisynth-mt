DuplicateFrame

DuplicateFrame (clip, int frame [, ...])

DuplicateFrame は DeleteFrame とは正反対のフィルタです。実引数として与えられた一連のフレームを複製します。DeleteFrame と同様、サウンドトラックは変更されません。

使用例:

DuplicateFrame(3, 3, 21, 42) # 4 フレーム追加

更新履歴:
