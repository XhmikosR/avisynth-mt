MessageClip

MessageClip (string message, int "width", int "height", bool "shrink", int "text_color", int "halo_color", int "bg_color")

MessageClip は、テキストメッセージを含むクリップを生成します。（AviSynth の）内部ではエラー報告に使用されています。
フォントは Arial が使用され、可能であれば width x height のクリップにフィットするように 24 ポイントから 9 ポイントまでの文字サイズから選択されます。
shrink が true なら、クリップの解像度は、必要に応じてテキストにフィットするように縮小されます。
