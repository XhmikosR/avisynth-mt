Informação Geral

A descrição de um plugin começa com alguma informação geral. Por exemplo

autor: Rainer Wittmann (aka kassandro)
versão: 0.6.1
download: http://www.removedirt.de.tf/
categoria: Suavizadores temporais
requisitos:

  • YV12 & YUY2 Colorspace
  • Suporte SSEMMX
  • a largura e a altura do clipe devem ser um múltiplo de 8
A primeira linha autor dá o autor(es) do plugin. Pode ser seu nome real, apelido de foro, ou ambos.
A segunda linha versão dá a versão do plugin na qual é descrita nesta documentação. Note que pode não ser a mais recente versão do plugin.
A terceira linha download dá a página onde baixar o plugin.
A quarta linha categoria, é a categoria na qual o plugin pode ser achado.
A última linha requisitos, são as exigências para poder usar o plugin. Elas podem ser o espaço de cor suportado (YV12 e YUY2 neste caso), um suportada de CPU (aqui de que é SSEMMX) ou que a largura / altura deve ser um múltiplo de algum número (normalmente 8 ou 16). O último é uma exigência para a otimização no plugin.

Uma nota final sobre a CPU que pode ser requerido para um plugin. Se você não sabe o tipo de instruções de otimização que é usada por sua CPU, observe na tabela seguinte:

Remoção de Desentrelaçamento & Pulldown

Todo os vídeos PAL, NTSC e SECAM são entrelaçados, o que significa, só uma linha é transmitida a cada intervalo de renovação.  Os filtros Deinterlacing o deixam tomar conta de qualquer problema causado por isso. Os filtros IVTC (telecine inverso, remoção de pulldown de aka) desfazem o processo telecine, que vem de diferenças entre o momento de seu vídeo e sua fonte original.

Pacote de Filtro Decomb (por Donald Graft) Este pacote das funções de plugin para AviSynth fornece os meios para remover os artefatos de fluxos progressivos telecinados, fluxos entrelaçados e misturas disso. As funções podem ser combinadas para implementar o telecine inverso para NTSC e PAL. [discussão].
DGBob (por Donald Graft) Este filtro divide cada campo da fonte em seu próprio quadro e de forma adaptável cria as linhas perdidas interpolando o campo atual ou usando os dados do campo anterior. [discussão].
FDecimate (por Donald Graft) Este filtro provê capacidades de dizimação estendidas não disponível no Decimate(). Pode remover quadros de um clipe para alcançar a taxa de quadro desejada, retendo a sincronização de áudio/vídeo. Remove preferencialmente os quadros duplicados onde possível. [discussão].
KernelDeint (por Donald Graft) Este filtro desentrelaça usando uma aproximação do kernel , núcleo. Dá resolução vertical grandemente melhorada em áreas desentrelaçadas, comparadas a campo simples de descarte. [discussão].
MultiDecimate (por Donald Graft) Remove N de todos os quadros M, levando os quadros mais semelhante aos seus anteriores. [discussão].
IBob (por Kevin Atkinson) Este filtro simples trabalha exatamente como o filtro interno Bob do Avisynth exceto que usa interpolação linear em vez de redimensionamento bicubic. [discussão].
SmartDecimate (por Kevin Atkinson) Este filtro remove telecine combinando seus campos e dizimação ao mesmo tempo, diferente do tradicional acesso para comparar quadros de telecine e remover duplicatas. [discussão].
GreedyHMA (por Tom Barry) O código do algoritmo Greedy/HM DScaler para executar pulldown comparando, filtrando e desentrelaçando vídeo. [discussão].

Filtro de desentrelaçamento de Movimento compensado "TomsMoComp" (por Tom Barry)

Este filtro usa compensação de movimento e processo adaptável para desentrelaçar fonte de vídeo (não para filme NTSC). [discussão].
UnComb IVTC (por Tom Barry) Filtro para comparar campos iguais e diferentes de fonte de vídeo de filme NTSC ou PAL corretamente telecinado. [discussão].

Suavizadores de Espaço Temporal

Estes filtros usam semelhanças e diferenças de cor, ambos dentro e entre quadros para reduzir ruído e melhorar o tamanho comprimido. Eles podem melhorar grandemente o vídeo ruidoso, mas um pouco de cuidado deve ser tomado com eles para evitar movimento borrado e perda de detalhe.

PeachSmoother (por Lindsey Dubb) Um suavizador adaptável aperfeiçoado para transmissões de TV. O Peach, (pêssego), trabalha procurando píxeis bons e juntando sua névoa laranja. Quando juntou bastante, espalha sobre os píxeis ruins, tornando-os melhores. [discussão].
FluxSmooth (por SansGrip) Os píxeis flutuantes são retirados da ocorrência, pelo cálculo médio com seus adjacentes. [discussão].
NoMoSmooth (por SansGrip) Um suavizador de movimento adaptável de espaço temporal. [discussão].
MipSmooth (por Sh0dan) Pega o quadro da fonte e cria três versões novas, cada um com metade do tamanho do anterior. Eles são re-escalados ao tamanho original. São comparados com o original e se a diferença estiver abaixo do percentual, a informação é usada para formar o píxel final. [discussão].
SpatioTemporal Median Filter "STMedianFilter" (por Tom Barry) STMedianFilter é um filtro espacial / temporal mediano (de movimento ligeiramente compensado).
Convolution3D / Convolution3DYV12 (por Vlad59) Convolution3D é um suavizador espacial / temporal, aplica um filtro 3D de convolução a todos os píxeis de quadros sucessivos. [discussão].

Suavizador Espacial

Estes usam semelhanças e diferenças de cor dentro de um quadro para melhorar a imagem e reduzir tamanho comprimido. Eles podem suavizar ruído muito bem, mas colocações agressivas demais neles podem causar uma perda de detalhe.

Smoother Masked  "MSmooth" (por Donald Graft) Este filtro é efetivo para remover ruído de mosquito como efetivamente também áreas planas suavizadas em anime. [discussão].
SmoothUV (por Kurosu) Este filtro pode ser usado para reduzir arco-íris, como feito por SmartSmoothIQ. [discussão].
VagueDenoiser (por Lefungus) Um anti-ruído simples que usa oscilações. [discussão].
Deen (por Marc FD) Vários anti-ruídos. Este filtro pode causar pouca memória, assim use a seu próprio risco. [discussão] 

Suavizadores Temporais 

Estes filtros usam semelhanças de cor e diferenças entre quadros para melhorar a imagem e reduzir o tamanho  comprimido. Eles podem limpar a maioria do ruído em áreas estacionárias sem perder detalhe, mas colocações fortes demais podem causar embaçamento das áreas de movimento.

TemporalCleaner (por Jim Casaburi; passado ao AviSynth por Vlad59) Um anti-ruído temporal simples mas muito rápido, que visa melhorar a compressibilidade.
RemoveDirt (por kassandro) Um limpador temporal com forte proteção contra artefatos. [discussão].
GrapeSmoother (por Lindsey Dubb) Quando as cores mudam só um pouco, o filtro decide que é provavelmente ruído e só muda ligeiramente a cor do quadro prévio. Quando a mudança de cor aumenta, o filtro fica cada vez mais convencido de que a mudança é devida ao movimento em vez de ruído e a nova cor adquire peso cada vez mais. [discussão].
Chroma Noise Reducer "Cnr2" (por Marc FD) Reduz o ruído no croma (UV) e preserva o luma (Y). [discussão].

Plugins para Aguçar/Suavizar

Estes são intimamente relacionados aos Suavizadores Espaciais, acima.  Eles tentam melhorar a qualidade da imagem aguçando ou suavizando as extremidades.

MSharpen (por Donald Graft) Este plugin para Avisynth implementa um conceito incomum em aguçar espaço. Embora especificamente projetado para anime, também trabalha muito bem em vídeo normal. O filtro é muito efetivo ao aguçar extremidades importantes sem ampliar o ruído. [discussão].
asharp (por Marc FD) Filtro de aguçamento adaptável. [discussão].
aWarpSharp (por Marc FD) Filtro aguçador de deformação.
Unfilter plugin (por Tom Barry) Este filtro suaviza/aguça um clip. Implementa filtros horizontais e verticais projetados para reverter (ligeiramente) os esforços prévios para melhor suavizar ou aguçar o controle em DVD que são comuns (mas feios). [discussão].
WarpSharp WarpSharp.
Xsharpen Este filtro executa um efeito de aguçamento sutil mas útil.

Redimensionadores

Plugins para redimensionar seu clipe.

BicublinResize (por Marc FD) Este é um jogo de amostras: FastBilinear (semelhante ao simpleresize de Tom Barry), FastBicubic (um exemplificador não filtrado Bicubic) e Bicublin (usa bi-cúbico em plano e bilinear Y em planos UV). [discussão].
SimpleResize (por Tom Barry) Duas simples e rápidas batidas de interpolação lineares  São não filtradas, não suavizarão muito.
YV12InterlacedReduceBy2 (por Tom Barry) InterlacedReduceBy2 é um filtro Reduz por 2 rápido, útil como um muito rápido redutor de um clipe entrelaçado. [discussão].

VSFilter (por Gabest) Possibilita importar vários formatos de subtítulos como *.sub, *.srt, *.ssa, *.ass, etc. [discussão].

Plugins Decodificadores MPEG (fonte) 

Plugins que possibilitam importar arquivos mpeg2 (inclusive hdtv transportáveis).

MPEG2Dec (por dividee e outros) Mpeg2dec é um plugin para AviSynth importar arquivos MPEG2. (saídas para YUY2)
MPEG2Dec3 (por Marc FD e outros) Uma modificação de MPEG2dec2.dll com deblocking e deringing. Note que a informação de colorspace de dvd2avi é ignorada ao usar mpeg2dec. [discussão].
Mpegdecoder (por Nic) Esta DLL carrega arquivos VOB/MPEG-2/MPEG-1 diretamente ao AviSynth. [discussão].

Plugins Decodificadores MPA (fonte)

Plugins que possibilitam importar arquivos  mpa.

MPASource (por Warpenterprises) Plugin decodificador de áudio mp1/mp2/mp3. [discussão].

Plugins para comparar a qualidade do vídeo

SSIM (por Lefungus) Filtro que compara a qualidade do vídeo (semelhante a psnr, mas usa uma qualidade diferente métrica de vídeo). [discussão].
VqmCalc (por Lefungus) Filtre para comparar a qualidade do vídeo (semelhante a psnr, mas usa uma qualidade diferente métrica de vídeo). [discussão].

Plugins de Vídeo de Radiodifusão

Significa ter cuidado sobre vários problemas que mostram quando os vídeos são capturados do ar.  Alguns ajudam na separação de luma/croma; outros reduzem problemas de interferência ou compensam o overscan.

AutoCrop plugin (por CropsyX) Apara  automaticamente as bordas pretas de um clipe. [discussão].
DeSpot (por Fizick) Este filtro é projetado para remover ruído temporal na forma de pontos (manchas) e listras de alguns vídeos. O filtro também é útil para restauração (limpeza) de velhos filmes de telecine 8mm (e outros) de manchas (de pó) e algumas faixas (arranhões). [discussão].
DeScratch (por Fizick) Este plugin remove arranhões verticais de filmes. [discussão].
Guava Comb (por Lindsey Dubb) Este é um filtro pente, pretende retirar arco-íris, rastos de ponto e cintilações em partes estacionárias de uma imagem. [discussão].
BorderControl (por Simon Walters) Depois de capturar o vídeo você pode querer apará-lo para se livrar de lixos.  BorderControl permite borrar bordas incluídas em vez de incluir bordas sólidas prevenindo artefatos entre quadro e bordas. [discussão].
FillMargins (por Tom Barry) Filtro similar a BorderControl. [discussão].
Reinterpolate411 (por Tom Barry) Parece aqueles píxeis de croma plano estão sendo duplicados no codec de MainConcept (NTSC). O novo filtro ajudará a descartar os píxeis de croma estranhos, recriando-os com a média de 2 píxeis adjacentes. [discussão].

Miscelânea de Plugins

Dup (por Donald Graft) Planejado para uso em clipes que têm um número significante de quadros de conteúdos duplicados, mas que difere devido ao ruído. Tipicamente anime tem muitas dessas tais duplicatas. Substituindo duplicatas ruidosas por outras exatas, pode-se obter uma redução de bitrate. [discussão].
DePan (por Fizick) As ferramentas DePan estimam movimento global (panorama) em quadros e faz compensação total ou parcial de movimento global. [discussão].
AviSynth monitor "avsmon" (por johann.Langhofer) Este plugin permite prever o vídeo durante a conversão e determinar o atraso exato do áudio. [discussão].
MaskTools (por Kurosu e Manao) Este plugin trata da criação, do aprimoramento e da manipulação de tal máscara para cada componente de colorspace de YV12. [discussão].
MVTools (por Manao) Coleção de filtros (Blur, ConvertFPS, Denoise, Interpolate Mask e outros) que usam vetores de movimento gerados por este plugin. [discussão].
VideoScope (por Randy French) Você pode usar este plugin para pintar as cores de um quadro. Mostra um monitor waveform (wfm) e um vectorscope. [discussão].
AudioGraph (por Richard Ling, modificado por Sh0dan) Exibe o formato do áudio em cima do vídeo. [discussão].
Blockbuster (por Sansgrip) Com este filtro pode-se usar vários métodos para reduzir ou eliminar blocos DCT: incluindo ruídos (distribuição Gaussian), aguçando ou borrando. [discussão].
ReverseFieldDominance (por Simon Walters) Reverte o domínio de campo PAL DV. [discussão].
ChromaShift (por Simon Walters) ChromaShift troca a informação de crominância em qualquer direção, para compensar o registro incorreto de Y/UV. [discussão].
AddGrain (por Tom Barry) AddGrain gera filme granulado ou outros efeitos (como chuva) acrescentando ruído aleatório a um clipe de vídeo. Este ruído pode opcionalmente ser combinado horizontal ou verticalmente para criar listras.
DctFilter (por Tom Barry) Reduz componentes de ruído de altas freqüências que usam Transformação de Co-seno Discreto e seu inverso.  Resulta em um alto ganho de compressão, quando usado no fim do seu script.  Altura/largura deve ser um múltiplo de 16. [discussão].
Undot (por Tom Barry) UnDot é um filtro mediano simples para remoção de pontos que são píxeis órfãos perdidos e ruído de mosquito.  Corta cada valor de píxel para ficar entre o mínimo e o máximo de seus oito circunvizinhos. [discussão].
DVinfo (por WarpEnterprises) Este filtro pega a info de data e hora de gravação de um DV-AVI. Deve trabalhar com Tipo-1 e Tipo-2, AVI normal e openDML. [discussão].
RawSource (por WarpEnterprises) Este filtro carrega dados de vídeos não tratados. [discussão].
