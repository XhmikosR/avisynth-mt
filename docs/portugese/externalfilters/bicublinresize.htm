Descrição

Este é um grupo de reamostradores/redimensionadores para avisynth 2.5 e todos aqui são não filtrados, significa que são rápidos demais, mas longe de perfeito, porque eles usam um número pequeno de pontos de amostra. De qualquer maneira, para tarefas normais, eles podem satisfazê-lo perfeitamente. ^^

BTW, é importante testar isto e deixar SEUS olhos decidirem. A razão é que até mesmo se não for interpolação real totalmente filtrada, parece muito decente de qualquer maneira.

Sintaxe

FastBilinearResize (h, v)
FastBicubicResize (h, v, "b", "c")
BicublinResize (h, v, "b", "c')

FastBilinear é como o resize simples de trbarry.
FastBicubic é um reamostrador não filtrado de Bicubic.
Bicublin use bicubic Y em plano e bilinear em planos UV.

h, v = resoluções horizontais e verticais.
b, c = coeficientes bicubic. Padrão: b = 1/3, c = 1/3
