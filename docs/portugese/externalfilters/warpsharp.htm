WarpSharp

Abstrato

autor: anônimo (japonês)
versão: em warpsharp_2003_1103.cab
download: http://www.geocities.co.jp/SiliconValley-PaloAlto/2382/ (english intro)
categoria: Plugins Sharpen/Soften
requisitos: YUY2 & YV12 Colorspace

Você também precisará copiar msvcp71.dll e msvcr71.dll ao seu diretório de sistema.

Descrição

A sintaxe é determinada por

WarpSharp (clip, int depth = 128, int blur = 3, int bump = 128, float cubic = -0.6)

Descrição dos parâmetros

depth
Profundidade de deformação .

blur
Número de vezes de borrar. O processo ficará lento se este valor aumenta.Se este valor não está definido, o valor é fixado ao mínimo, 1.

bump
Percentual de detecção de desigualdade. O efeito se tornará mais aguçado se aumentar.

cubic
Coeficiente do valor de deformação (interpolação cúbica). Nenhuma necessidade de mudar esta colocação.
