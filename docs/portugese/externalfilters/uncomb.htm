Descrição

UnComb é um simples filtro IVTC para comparar até mesmo campos diferentes de filme de fonte de vídeo NTSC ou PAL corretamente telecinados. Pense nisso como um pobre irmãozinho para suporte ao Telecide(). É rápido porque não tenta achar quadros para desentrelaçar.

USO - Para usá-lo apenas

Em seu arquivo Avisynth use comandos semelhantes para

LoadPlugin("F:\UnComb\UnComb.dll")
AviSource("D:\wherever\myfile.avi")
UnComb()

Com certeza substitui os nomes de arquivo e diretório com o seu próprio. Não há nenhum parâmetro.

ASSUNTOS CONHECIDOS E LIMITAÇÕES

1) Requer Avisynth 2.5 e entradas YV12.

2) Desculpe, atualmente requer um P-III, Athlon, ou mais alto. Necessita suporte ao SSEMMX.

3) Apenas faz a melhor comparação de campo disponível. Não fará mistura, interpolação ou desentrelace. É para material de fonte razoavelmente boa.

4) Não faz e provávelmente não terá opções especiais como Telecide(). Use Telecide() quando necessário.

5) Não dizima. Siga UnComb com Decimate(5) quando você precisa dizimar quadros de 30 para 24 fps.

6) É rapidamente malvado. ;-)
