Descrição

Este filtro limpa o filme, tirando a média do ruído visual entre quadros

Quando as cores mudam só um pouco, o filtro decide que é provavelmente ruído e só muda ligeiramente a cor do armação quadro. Como a mudança de cor aumenta, o filtro é convencido cada vez mais que a mudança se deve a movimento e não de ruído - e a nova cor adquire cada vez mais peso.

Para usar o GrapeSmoother, seu computador terá que ser apto a rodar instruções MMX. Isso significa um computador mais recente que o Pentium I original. Roda muito depressa.

Este filtro trabalha no colorspace YUY2. Se o clipe é codificado com cores RGB, você precisará primeiro processar com ConvertToYUY2().

Sintaxe

GrapeSmoother (clip, int "smoothing")

Smoothing
Um inteiro de 1 a 100; 30 por padrão

Esta é a única opção. Quanto mais alto fixar isto, mais ruídos você cortará - mas verá mais borrão. Em geral, gama razoável de valores de 10 (para suavizar a mancha ocasional sem artefatos) para 25 (reduzir ruído com artefatos mínimos) para 45 (para melhor redução, mas obscurecendo visivelmente) até 80 (calcular a média de ruído realmente pesado às custas de obscurecer mal). A valores extremamente altos para smoothing, você começará a ver linhas de artefatos verticais.

Para escolher uma colocação para isto, tente em uma comédia ou drama - qualquer coisa com close-up de faces que se movem. Aumente smoothing até a supressão não parecer o valor do obscurecer.
