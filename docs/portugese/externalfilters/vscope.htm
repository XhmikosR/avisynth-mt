Sintaxe

VideoScope (clip, string DrawMode, bool TickMarks, string HistoTypeSide, string HistoTypeBottom, string FrameType)

DrawMode: pode ser "side", "bottom", ou "both"

TickMarks: true ou false

HistoTypeSide / HistoTypeBottom : "Y", "U", "V", "UV", ou "YUV"

FrameType: "blank", "colormap", "Y", "U", "V", "UV"
(só visto quando drawmode é "both")

VideoScope só trabalha com vídeo YUY2. Para usar com RGB inclua o comando 'ConvertToYUY2 ' na frente.

O que ele faz

Você pode pensar em VideoScope como um super 'histográfico'. Em vez de puxar um histográfico para o quadro inteiro, puxa um para cada linha (lado), ou cada coluna (fundo).

Como um exemplo digamos que você usou o comando:

VideoScope("bottom", true, "Y", "Y", "Y")

Você verá o gráfico sob o quadro. A posição vertical no gráfico mostra o valor Y (brilho) dos píxeis no quadro. O brilho dos píxeis no gráfico mostra quantos píxeis tiveram aquele valor Y. Se só um píxel tem aquele valor então será escuro (mas ainda visível). Se muitos píxeis têm aquele valor, será branco luminoso. O gráfico tem 256 píxeis de altura desde que estejam lá 256 possíveis valores Y.

Você pode usar marcas de tiques para conseguir uma leitura bem exata dos valores Y.
Os tiques azuis são 0-63 e 128-191.
Os tiques marrons são 64-127 e 192-255.
Os tiques verticais amplamente espaçados são separados por 16.
Os tiques espaçados mais pertos são separados por 8.
os tiques triplos são separados por 4.

VideoScope("both", false, "Y", "Y", "Y")

No fundo do canto direito o histograma total do quadro. É semelhante ao histograma do VirtualDub. Porém usa escala linear ao passo que VirtualDub parece usar uma escala logarítmica.

VideoScope("both", true, "U", "V, "UV")

O lado (roxo) gráfico mostra o valor U.
O fundo (verde) gráfico mostra o valor V.
A exibição de interesse de todo o quadro é o mapa UV. A cor de fundo é igual a 'colormap' que mostra toda a gama UV.
Y se mantém constante em 128.
V é vertical (0-255 do fundo ao topo).
U é horizontal (0-255 da esquerda para a direita).
O retículo representa o valor neutro (cinza) 128,128.
Os valores UV de todos os píxeis estão traçados no gráfico. Se um só píxel tiver um valor UV particular, será exibido como um ponto preto. Se muitos píxeis têm esse valor, será mais luminoso.
