AutoCrop

Abstrato

autor: Glenn Bussell
versão: 1.1
download: http://www.videofringe.com/autocrop
categoria: Plugins de Vídeo de Radiodifusão
requisitos: YV12 & YUY2 Colorspace

Descrição

AutoCrop é um filtro AVISynth que apara automaticamente as bordas pretas de um clipe. Opera em qualquer modo de prévia onde sobrepõe a informação de aparo indicada sobre o clipe existente, ou modo aparo onde realmente apara o clipe. Também pode assegurar que a largura e a altura são múltiplos de números especificados, assim o clipe aparado pode ser passado ao compressor de vídeo de sua escolha sem problemas.

Uso

AutoCrop tem 10 parâmetros, todos opcionais, assim você pode apenas fazer AutoCrop() para saber como trabalha o filtro. Podem ser especificados parâmetros em ordem ou por nome. Eu sugiro especificar por nome por ter muitos parâmetros!

A ordem dos parâmetros segue:

AutoCrop ("mode", "wMultOf", "hMultOf", "leftAdd", "rightAdd", "topAdd", "bottomAdd", "threshold", "samples", "aspect")

Sintaxe

mode - Inteiro - padrão 1

0 - Crop - Apara a imagem
1 - Preview - A informação de aparo sugerida é sobreposta ao clipe existente, inclusive um comando de aparo que você pode usar para substituir AutoCrop.
2 - Log - Anota os parâmetros de aparo no arquivo "AutoCrop.log" no diretório atual.

wMultOf - Inteiro - padrão 4

Assegura que a largura do clipe aparado será um múltiplo do número especificado. Use isto para assegurar que o clipe é válido para qualquer codec a comprimir.

hMultOf - Inteiro - padrão 2

Como acima, mas para altura.

leftAdd - Inteiro - padrão 0

Força autocrop para aparar e píxeis adicionais leftAdd depois de completar a auto detecção. Você pode usar esta opção junto com wMultOf. Útil para aparar linhas de qualidade ruins ao lado de capturas de VHS.

rightAdd - Inteiro - padrão 0

Veja acima, mas para a direita.

topAdd - Inteiro - padrão 0

Força autocrop aparar e píxeis adicionais topAdd depois de completar a auto detecção. Você pode usar esta opção junto com hMultO. Útil para aparar linhas de qualidade ruins no topo de capturas de VHS.

bottomAdd - Inteiro - padrão 0

Veja acima, mas para o fundo.

threshold - Inteiro (0-255) - padrão 40

Percentual é a média de luminância comum que uma linha deve ter antes de ser considerada não em branco.

samples - Inteiro - padrão 10

O número de quadros para examinar ao determinar a informação de aparo. Isto afeta diretamente o início da cronometragem do filtro. Em geral 10 parece ser bastante, se o início lento o está aborrecendo abaixe o número.

aspect - Flutuante - padrão 0 - Aspeto é ignorado

-1 - Mantém a relação de aspecto do clipe fonte
0 - Aspeto é ignorado
>0 - A relação de aspecto será fixada nisto. Por exemplo para uma relação de aspecto 4:3 passe 4.0/3.0 ou 1.33333333333. Note que 4/3 não trabalha. O .0 é preciso para contar ao AviSynth que o valor é um flutuante.

Uso Recomendado

Eu sugiro fortemente o uso do modo prévia antes de executar um aparo em particular. Também é digno notar que  copiar as colocações sugeridas no comando de aparo normal AviSynth é mais rápido que rodar o AutoCrop em mode = 0.
