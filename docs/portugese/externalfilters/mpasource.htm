Descrição

Abre arquivos mp1/2/3 de áudios diretamente e normaliza a produção a 100% se pedido.

Tome cuidado ao testar: abrindo o escrito AVS o arquivo inteiro é escaneado para retornar o número exato de amostras quando normalize=false (padrão) é feito com decodificação parcial (±20 segundos para uma hora de áudio em meu PC). Com normalize=true, é muito mais lento (±100 segundos).

A versão 0.2 cria um arquivo pequeno .D2A depois do primeiro escaneio, assim não é necessário muitos escaneios.

Quando são pedidos saltos para trás, a busca é sempre feita em modo rápido do quadro 0 para o quadro pedido.

Junto com o áudio, um pequeno vídeo 8x8 é gerado, assim você sempre tem que pôr MPASource como SEGUNDO argumento em AudioDub.

Exemplo 

LoadPlugin("C:\myprojects\mpasource25\release\mpasource.dll")
V = BlankClip(height=100, width=100, length=20000, fps=25)
A = MPASource("d:\_mp3\pop\liquido - narcotic.mp3", normalize = false)
AudioDub(V, A)
