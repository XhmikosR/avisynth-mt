Descrição

Basicamente, este filtro é uma máscara de não aguçar muito comum,

simplesmente porque essa máscara é a técnica

de aguçamento mais agradável para meus olhos ^ ^

mas incluí 3 idéias simples que tive sobre aguçar:

- percentual adaptável para evitar aumento de ruído
- aguçamento adaptável de bloco para evitar aumento das extremidades dos bloco DCT
- aguçamento adaptável baseado em bloco para evitar aumento das extremidades dos blocos DCT

é aperfeiçoado para qualidade, não velocidade. mas... eu amo velocidade ^ ^
assim incluí o código MMX/ISSE muito aperfeiçoado.

está trabalhando em colorspace YV12, e em luma só
(aguçamento do croma parece só introduzir artefatos)

Uso do asharp

Primeiro, inclua a linha seguinte em seu escrito.avs: 
LoadPlugin("asharp.dll") # porque Avisynth 2.5 ainda não suporta Autoloading

Para fazer não aguçar simples que mascara com uma força de 2x:

Asharp(2,0)

Tentar aguçamento adaptável:

Asharp(2, 4)

Sugestão para decodificação de anime divx:

Asharp(2. 5, 4.5, 0.25, hqbf=true)

Sintaxe

asharp (float "T", float "D", float "B", bool "hqbf")

T não aguçar que mascara percentual. 0 não fará nada. (valor preso em [nothing=0.. 32])

T = 1 é como 32 com filtro máscara de não aguçar ffdsow/Vdub, aumentará contraste 1x. padrão é T = 2.

D : força de aguçamento adaptável. (valor preso [desativado=0.. 16])

fixe em 0 para desabilitar.
Se D > 0, percentual adaptável é ativado.
O percentual é adaptado para cada píxel (maior para extremidades).
Se aguçamento adaptável é habilitado, T atua como máximo.
padrão é D = 4.

B : aguçamento de bloco adaptável. (valor preso [desativado=0.. 4])

Fixe um valor negativo para desativar.
Se B >= 0, aguçamento de bloco adaptável é habilitado.
Age muito simplesmente, abaixando o percentual ao redor de extremidades de blocos DCT.
Se você usar, evite qualquer aparo não mod8 entre o decodificador e asharp.
(só trabalha com aguçamento adaptável, quando D > 0)
padrão é B = -1 (desativado)

hqbf : filtro de bloco de alta qualidade. (aka percentual adaptável com base em bloco).

Era difícil implementar em SIMD, mas eu adoro.
Tente com e sem em um vídeo maciço e você entenderá por que ^^
Só trabalha com percentual adaptável.
padrão é false (desativado)
