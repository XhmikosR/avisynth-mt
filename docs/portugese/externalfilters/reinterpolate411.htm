Descrição

Este filtro só está bem testado com o decodificador de MainConcept. Mas parece que os píxeis de croma planos apenas são duplicados naquele codec. Este filtro ajudará isso descartando os píxeis croma estranhos e os recriando na média dos 2 píxeis planos adjacente.

Não importa se o material é entrelaçado. Não há nenhum parâmetro, só suporte ao YUY2 de Avisynth 2.5. Provavelmente use diretamente depois de AviSource como:

AviSource(...)
ReInterpolate411()
