Descrição

É pretendido que este filtro inverta o domínio de campo (eu me refiro à leitura aqui para uma melhor explicação da ordem de campo na web) sobre vídeo DV PAL.

Minha Matrox G400 com saída para TV produz primeiro o campo de topo de um avi e depois o campo de fundo.

Mas o DV PAL é sempre armazenado primeiro como campo de fundo e este filtro inverte a ordem de campo simplesmente movendo cada linha para cima (ou para baixo) por uma linha e duplicando a inferior (ou a superior).

Confirme este assunto no Doom9 para ver como ocorreu e obter a versão do VirtualDub de Donald Graft.

Sintaxe

ReverseFieldDominance (bool shiftup = true)
