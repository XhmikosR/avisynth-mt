Trim

Trim (clip, int first_frame, int last_frame)

Trim apara um clipe de vídeo de forma a incluir só os quadros first_frame até o last_frame. O áudio é aparado igualmente de forma a ficar sincronizado. Se você passa 0 para last_frame significa "fim de clipe". Um valor negativo de last_frame retorna os quadros de first_frame até first_frame + (- last_frame-1). Este é o único modo para obter o primeiro quadro!

  

Trim(100,0)     # apague os primeiros 100 quadros
Trim(100,-100)  # igual a trim(100,199)
Trim(0,-1)      # retorna só o primeiro quadro
