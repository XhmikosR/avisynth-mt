SuperEQ

SuperEQ (clip, string filename)

Shibatch Super Equalizer é um plugin equalizador gráfico e paramétrico para Winamp. Este plugin usa a 16383ª ordem do filtro FIR com algoritmo de FFT. Sua equalização é muito precisa e pode ser fixada separadamente para cada canal. SuperEQ é escrito originalmente por Naoki Shibata.
SuperEQ requer um arquivo que fixa o equalizador foobar2k. O equalizador pode ser achado no foobar's DSPManager e os ajustes também são salvos lá.

Algumas colocações e ajustes podem ser baixados daqui.

Este plugin é aperfeiçoado para processadores que têm cache igual ou maior que 128k bytes (16383*2*sizeof(float)=128k). Este plugin não trabalha eficazmente com processadores da série K6 (compre Athlon!).

Parâmetro:

filename O arquivo equalizador prefixado foobar2k para aplicar.

Exemplo:

Aplique o filtro "Loudless" do prefixado equalizador acima:

SuperEq("C:\Equalizer Presets\Loudness.feq")
