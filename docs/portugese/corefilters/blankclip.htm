BlankClip / Blackness

BlankClip (clip "template", int "length", int "width", int "height", string "pixel_type", float "fps", int "fps_denominator",
   int "audio_rate", bool "stereo", bool "sixteen_bit", int "color", int "color_yuv")
Blackness ()

O filtro BlankClip produz uma cor sólida, clipe de vídeo silencioso do comprimento especificado (em quadros). O clipe passado como um argumento é usado como um modelo para taxa de quadro, tamanho de imagem e assim por diante, mas você pode especificar todas as propriedades do clipe sem ter que fornecer um modelo. Cor deve ser dada com valores RGB hexadecimais. Sem qualquer argumento, BlankClip produzirá um clipe negro de 10 segundos (RGB32), 640x480, 24 fps, 16 bit, mono de 44100 Hz.

template: se presente, o clipe resultante terá as propriedades do modelo (bug: exceto formato de cor), com exceção das propriedades que você define explicitamente.

length: comprimento do clipe resultante (em quadros).

width, height: largura e altura do clipe resultante.

pixel_type: tipo do píxel do clipe resultante, que pode ser " RGB24 ", RGB32 ", YUY2 " ou " YV12 ".

fps: o framerate (taxa de quadro) do clipe resultante.

fps_denominator: você pode usar esta opção se "fps" não for muito preciso. Por exemplo: fps = 30000, fps_denominator = 1001 (relação = 29.97) ou fps = 24000, fps_denominator = 1001 (relação = 23.976). É 1 por padrão.

audio_rate: samplerate (taxa de amostra) do áudio silencioso do clipe.

stereo: (boolean) quando fixado para true o áudio silencioso é em estéreo, quando fixado false uma trilha silenciosa mono é incluída

sixteen_bit: (boolean) true e false resultam 16 bit, exceto quando um modelo é usado onde um bit diferente é fixado.

color: especifica a cor do clipe, preto (= $000000) por padrão. Veja ColorPresets para cores prefixadas.

color_yuv: incluído na v2.55 e deixa especificar a cor do clipe com valores YUV. Requer colocação pixel_type para " YUY2 " ou " YV12 ", caso contrário não faz nada.

Blackness é um aliás para BlankClip, fornecido para retôrno de compatibilidade.

  

# produz um clipe preto (3000 quadros, largura 720, altura 576, framerate 25), com uma trilha de áudio silencioso (16 bit , 44.1 kHz estéreo):
BlankClip(length=3000, width=720, height=576, fps=25, color=$000000)

# produz um clipe preto (3000 quadros) com as propriedades restantes do clipe avi:
AviSource("E:\pdwork\DO-Heaven.AVI")
BlankClip(length=3000, color=$000000)

# põe um fluxo de áudio silencioso (com um samplerate de 48 kHz) para um clipe de vídeo:
video = AviSource("E:\pdwork\DO-Heaven.AVI")
audio = BlankClip(video, audio_rate=48000)
AudioDub(video, audio)
