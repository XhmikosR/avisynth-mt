SelectRangeEvery

SelectRangeEvery (clip, int "every", int "length", int "offset", bool "audio")

Este filtro está disponível a partir da v2.5. O filtro seleciona length número de quadros every n quadros, a partir do quadro offset. Valores padrões são: Selecione 50 quadros a cada 1500 quadros, começando com a primeira seleção no quadro 0. (every = 1500, length = 50, offset = 0).

A partir da v2.55, SelectRangeEvery também processa áudio. Para manter o original, use, audio = false.

Exemplos: 

# Selecione os quadros 0 a 13, 280 a 293, 560 a 573, etc.
SelectRangeEvery(clip, 280, 14)

# Selecione os quadros 2 a 15, 282 a 295, 562 a 575, etc.
SelectRangeEvery(clip, 280, 14, 2)
