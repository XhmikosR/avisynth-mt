Compare

Compare (clip_filtered, clip_original, string "channels", string "logfile", bool "show_graph")

Compare produz o clipe filtrado (primeiro arg) com texto (Significa Absoluta, Significa Diferença & PSNR) e o gráfico de PSNR de Marc (se show_graph é true). A partir da v2.53 também o 'Overal PSNR' é mostrado na tela, mas não no logfile. A string de canais é uma combinação de R,G,B [,A] ou Y,U,V, dependendo dos clipes fontes. A string vazia (padrão) significa "YUV" ou "RGB". Se logfile não for nulo, os resultados serão escritos a um arquivo por este nome e não serão levados ao clipe. É muito mais rápido se você precisar comparar muitos quadros.

  

# Exibe diferenças na tela
Compare(clip1, clip2)
# para criar um arquivo log:
Compare(clip1, clip2, "", "compare.log")
# comparará só canais cromas de clipes YUY2.
Compare(clip1, clip2, "UV")
