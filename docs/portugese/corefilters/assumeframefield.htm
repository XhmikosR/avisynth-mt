AssumeFrameBased(clip clip)

AssumeFieldBased(clip clip)

AviSynth lembra se um determinado clipe é baseado em campo ou em quadro. Se o clipe é base campo também se lembra da paridade de cada campo (quer dizer, se é o topo ou o campo de fundo de uma quadro). Se o clipe é base quadro se lembra do campo dominante em cada quadro (quer dizer, qual campo no quadro vem primeiro quando eles estiverem separados).

Porém, esta informação não está necessariamente correta, porque normalmente não é armazenada apenas a informação de campo em arquivos de vídeo e os filtros fonte do AviSynth adivinha isto. AssumeFrameBased e AssumeFieldBased contam para AviSynth o tipo correto de um clipe.

AssumeFrameBased elimina a informação existente e assume que o clipe é base quadro, com o (até) campo do fundo dominante em cada quadro. (Isto acontece para ser o que os filtros fontes supõe.) Se você quiser o campo de topo dominante, use ComplementParity depois.

AssumeFieldBased elimina a informação existente e assume que o clipe é base campo, com os campos numerados que são campos de fundo e os campos de números diferentes como campos de topo. Se você quiser isto ao contrário, use ComplementParity depois.
