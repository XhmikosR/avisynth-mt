ShowFiveVersions

ShowFiveVersions (clip1, clip2, clip3, clip4, clip5)

ShowFiveVersions pega cinco fluxos vídeos e os combina em um arranjo cambaleante da esquerda para a direita.O único uso para isto (eu acho) é ajudar achar o pulldown NTSC padrão. Você pode fazer este código assim: 

# Visualizar todos os cinco padrões de pulldown de uma vez
DoubleWeave()
# ponha aqui um filtro de resizing se necessário (veja abaixo)
a = Pulldown(0,2).Subtitle("0,2")
b = Pulldown(1,3).Subtitle("1,3")
c = Pulldown(2,4).Subtitle("2,4")
d = Pulldown(0,3).Subtitle("0,3")
e = Pulldown(1,4).Subtitle("1,4")
ShowFiveVersions(a,b,c,d,e)

Este código exibe os cinco padrões de pulldown com algum texto que identifica qual é qual. Eu olho então pelo filme e escolho o padrão que evita a mistura de quadros. (Em pulldown normal, haverá na verdade dois que trabalham igualmente bem. Olhe para os diagramas na seção de filtros Pulldown para ver por que).  Se nenhum dos cinco trabalham, então você está lidando com uma das formas mais perversas de pulldown e pode querer usar PeculiarBlend.

A propósito, se você estiver planejando capturar a uma resolução alta e então reduzir, como recomendei em outro lugar, você deve provavelmente colocar o ReduceBy2 ou BilinearResize logo após a declaração de DoubleWeave no código acima. Antes de DoubleWeave não trabalhará corretamente e se você adia, ShowFiveVersions produzirá um quadro realmente grande.
