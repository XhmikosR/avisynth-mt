ColorBars

ColorBars (int "width", int "height", string "pixel_type")

O filtro ColorBars produz um vídeo clipe que contém barras coloridas SMPTE escaladas a qualquer tamanho de imagem. O clipe produzido é 640x480, RGB32 [16,235], 29.97 fps, com 1 hora.

Um tom de teste também é gerado. O tom é um seno de 440Hz a 48KHz, 16 bit, estéreo. O tom pulsa no alto-falante DIREITO, sendo desligado e religado a cada segundo.

Na v2.56, pixel_type = "YUY2"  ou "YV12" é incluído (padrão "RGB32"). Note, isto é quase equivalente a

ColorBars().ConvertToYUY2(matrix="PC.601")  # não escalar a gama luma

Ao gerar diretamente dados de formato YUV as transições de cor são organizadas para ocorrer num limite de croma alinhado.

Nota: As versões YUV deste clipe contêm valores de píxeis menos que 16 (InfraBlack)
