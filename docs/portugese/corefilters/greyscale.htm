Greyscale

Greyscale (clip [, string "matrix"])

Convertido para cinzento (sem mudar o formato de cor).

A conversão é feita com coeficientes Rec.601. Na v2.56, é incluída a opção matrix. Ao fixar matrix="Rec.709", o clipe é convertido a cinzento com coeficientes Rec.709. Veja ColorConversions para uma explicação destes coeficientes. Ao fixar matrix="Average" (só para clipes RGB) o luma é calculado como (R+G+B)/3.
