DeleteFrame

DeleteFrame (clip, int frame)

DeleteFrame apaga um único quadro, determinado como argumento. A trilha sonora não é modificada, assim se você usa muito este filtro, pode ter grande assincronia.

Se você quer apagar uma faixa de quadros (a até b, digamos) junto à porção correspondente da trilha sonora, pode fazê-lo com Trim assim:

Trim(0,a-1) ++ Trim(b+1,0)
