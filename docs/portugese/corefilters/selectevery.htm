SelectEvery

SelectEvery (clip, int step_size, int offset1 [, int offset2 [, ...]])

SelectEvery é uma generalização de filtros como SelectEven e Pulldown. Eu acho que o modo mais fácil para descrever isto é por um exemplo: 

SelectEvery(clip,2,0)      # idêntico a SelectEven(clip)
SelectEvery(clip,2,1)      # idêntico a SelectOdd(clip)
SelectEvery(clip,10,3,6,7) # seleciona quadros 3, 6, 7, 13, 16, 17, ... do clipe fonte
SelectEvery(clip,9,0)      # seleciona quadros 0, 9, 18, ... do clipe fonte
E como sobre isto: 
# Pegue um clipe de entrada de 24 fps progressivo e aplique pulldown 3:2
# criando um clipe 30 fps entrelaçado
AssumeFrameBased
SeparateFields
SelectEvery(8, 0,1, 2,3,2, 5,4, 7,6,7)
Weave
