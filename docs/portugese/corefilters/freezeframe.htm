FreezeFrame

FreezeFrame (clip, int first_frame, int last_frame, int source_frame)

Os filtros FreezeFrame substituem todos os  first_frame e last_frame pela cópia de source_frame. A trilha não é modificada. Isto é útil para encobrir defeitos num vídeo em caso de você ter um quadro semelhante e sem defeito disponível.
