PeculiarBlend

PeculiarBlend (clip, int cutoff)

Este filtro mistura cada quadro com o seguinte de um modo peculiar. A porção do quadro embaixo da linha de escaneio (cutoff)th fica inalterada. A porção de cima (cutoff-30)th é substituída com a porção correspondente do quadro seguinte. As30 linhas de escaneio entre elas são misturadas incrementalmente para disfarçar o switchover.

Você provavelmente está se perguntando por que alguém usaria este filtro. Bem, é assim. A maioria dos vídeos que eram originalmente filme com a técnica 3:2 de pulldown está na descrição do filtro de Pulldown. Mas algum uso um sistema muito mais sórdido no qual o crossover para a próximo quadro acontece no meio de um campo - em outras palavras, campos individuais se parecem um quadro de filme no topo e outro embaixo. Este filtro desfaz parcialmente este efeito peculiar. Deveria ser usado depois de th antes do filtro Pulldown. Determinar o cutoff, examine um quadro que está misturado deste modo e fixe cutoff para o número da primeira linha de escaneio nos quais você nota uma mistura.

Este filtro só trabalha com entradas YUY2.
