Dissolve

Dissolve (clip1, clip2 [, ...], int overlap, float "fps")

Dissolve é como AlignedSplice, exceto que os clipes são combinados com algumas sobreposições. Os últimos quadros overlap do primeiro vídeo são progressivamente misturados com os primeiros do segundo vídeo de forma que se fundem um ao outro. Os áudios são igualmente misturados.

Com clipes só de áudio, por padrão, overlap é em unidades de 1/24 de segundo. Fixe Fps=AudioRate() se é requerido o posicionamento exato da amostra do áudio.

O termo "dissolve" às vezes é usado para um efeito diferente no qual a transição é ponto mágico em vez de graduado. Este filtro não fará isso.
