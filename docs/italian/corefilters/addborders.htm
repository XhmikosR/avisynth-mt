AddBorders

AddBorders (clip, int left, int top, int right, int bottom, int "color")

AddBorders aggiunge attorno all'immagine dei bordi neri (o di altro colore), di una larghezza specificabile in pixels.

Per le controindicazioni nell'uso di formati colore differenti, si veda Convert

Il parametro color è opzionale (aggiunto nella v2.07), default=0 <NERO>, ed è specificato come valore RGB indipendentemente dal fatto che il formato del clip sia RGB o YUV.

Fate attenzione al fatto che molti algoritmi di compressione "lossy" non gestiscono bene bordi di colore uniforme a meno che la larghezza del bordo sia un multiplo del block size (16 pixels per MPEG).

Si può usare questo filtro in combinazione con Crop per spostare una immagine senza cambiare la dimensione del frame. Ad esempio: 

# # Sposta un'immagine 352x240 a destra di 2 pixels
Crop(0,0,350,240).AddBorders(2,0,0,0)
