SelectEvery

SelectEvery (clip, int step_size, int offset1 [, int offset2 [, ...]])

SelectEvery è una generalizzazione di filtri come SelectEven e Pulldown. Il modo migliore per spiegare l'uso di questo filtro è fare degli esempi: 

SelectEvery(clip,2,0)      # identico a SelectEven(clip)
SelectEvery(clip,2,1)      # identico a SelectOdd(clip)
SelectEvery(clip,10,3,6,7) # seleziona i frames 3, 6, 7, 13, 16, 17, ... 
SelectEvery(clip,9,0)      # seleziona i frames 0, 9, 18, ...
Ed ora questo: 
# Ad un clip progressive 24fps applichiamo un pulldown 3:2 ,
# producendo un output interlacciato a 30fps
AssumeFrameBased
SeparateFields
SelectEvery(8, 0,1, 2,3,2, 5,4, 7,6,7)
Weave
