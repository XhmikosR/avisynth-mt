SuperEQ

SuperEQ (clip, string filename)

Shibatch Super Equalizer è un plugin equalizzatore grafico e parametrico per winamp. Questo plugin usa un filtro FIR con algoritmo FFT di ordine 16383mo. La sua equalizzazione è molto precisa. L'impostazione dei parametri di equalizazione può essere fornita separatamente per ogni canale. SuperEQ è stato scritto da Naoki Shibata.

SuperEQ necessita di un file di impostazione foobar2k . Questo equalizer si può trovare nel DSPManager di foobar, così come le impostazioni.

Alcune impostazioni preimpostate si possono scaricare da qui.

Questo plugin è ottimizzato per processori che abbiano una cache > o = a 128k bytes (16383*2*sizeof(float) = 128k). Questo plugin non lavora efficacemente con i processori serie K6 (compratevi un Athlon!!!).

Parametri:

filename Il file di preset con le opzioni del foobar2k equalizer da applicare.

Esempi:

Applica il filtro "Loudless" dal file di Equalizer Presets :

SuperEq("C:\Equalizer Presets\Loudness.feq")
