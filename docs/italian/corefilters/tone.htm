Tone

Tone (float "length", float "frequency", int "samplerate", int "channels", string "type")

Questo genera un sound ad una data frequency per una data length in secondi. Type può essere "Silence", "Sine" (default), "Noise", "Square", "Triangle" or "Sawtooth".

Defaults sono Tone(10.0, 440, 48000, 2, "sine").
