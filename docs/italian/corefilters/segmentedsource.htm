SegmentedAVISource / SegmentedDirectShowSource

SegmentedAVISource (string base_filename [, ...], bool "audio", string "pixel_type")
SegmentedDirectShowSource (string base_filename [, ...] [, int fps])

ThIl filtro SegmentedAVISource carica automaticamente fino a 100 files avi per argomento (usando AVISource) e li lega fra loro (usando UnalignedSplice). Se è passato come argomento "d:\filename.ext" saranno caricati i files d:\filename.00.ext, d:\filename.01.ext e così via fino a d:\filename.99.ext . Qualunque file della sequenza che non esista sarà saltato.

SegmentedDirectShowSource funziona allo stesso modo.

Dalla v2.04 in avanti c'è incorporato il supporto per audio ACM (Audio Compression Manager) (es. mp3-AVIs).
