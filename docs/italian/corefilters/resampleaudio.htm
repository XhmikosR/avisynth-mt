ResampleAudio

ResampleAudio (clip, int new_sample_rate)

ResampleAudio esegue una modifica alla frequenza di campionamento audio utilizzando un algoritmo ad alta qualità.
La conversione non ha luogo se il samplerate è già quello voluto.

A partire dall v2.53 ResampleAudio accetta qualunque numero di canali.

  

# resamples audio to 48 kHz
source = AviSource("c:\audio.wav")
return ResampleAudio(source, 48000)
