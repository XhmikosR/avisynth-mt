Limiter

Limiter (clip, int "min_luma", int "max_luma", int "min_chroma", int "max_chroma")

Questo filtro è presente in v2.5. Lo standard conosciuto come CCIR-601 definisce il range del valore dei pixels considerato lecito per output su TV. Questi ranges sono 16-235 per la componente luma e 16-240 per quella chroma.

I Pixels al di fuori di questi range possono causare problemi con alcuni TV sets, ed è quindi meglio rimuoverli prima di encodare se si ha intenzione di usare preferibilmente la TV. Per default questo filtro taglia (o "fissa") i pixels sotto 16 a 16 e quelli sopra 235 (o 240) a 235 (o 240).

Prima della v2.53 il valore di default era 236 (che è sbagliato). Si usi Limiter(16, 235, 16, 240) per video digitali CCIR-601 compatibili.
