Loop

Loop (clip, int "times", int "start", int "end")

Ripete il segmento a partire dal frame start al frame end un dato numero di times.
times (default -1) è nil numero di volte che il loop deve essere applicato.
start (default 0) frame di inizio.
end (default framecount(clip)) frame di fine loop.

  

# Loops frame 100 to 110 of the current clip 10 times
Loop(10,100,110)


Loop() # make the clip loop (almost) endlessly


Loop(10) # make the clip loop ten times


Loop(10,20,29) # ripete i frames dal 20 al 29 10 volte prima di procedere, la durata del clip aumenta di 90 frames

Loop(0,20,29) # cancella i frames dal 20 al 29
              # la durata del clip diminuisce di 10 frames

Loop(-1,20,29) # frame 20 to 29 is repeated (almost) infinite times
