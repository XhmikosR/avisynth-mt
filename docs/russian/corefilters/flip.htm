FlipHorizontal / FlipVertical

FlipHorizontal (clip)
FlipVertical (clip)

FlipVertical переворачивает изображение вверх ногами (бывает полезен для работы с некоторыми кодеками, выдающими перевёрнутое изображение).
Аналогично, FlipHorizontal (появившийся в версии v2.5) меняет местами лево-право.
