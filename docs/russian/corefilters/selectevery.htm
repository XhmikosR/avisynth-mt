SelectEvery

SelectEvery (clip, int step_size, int offset1 [, int offset2 [, ...]])

SelectEvery это обобщение фильтров типа SelectEven и Pulldown. Думаю простейший путь описать его - примером:

SelectEvery(clip,2,0)      # идентично SelectEven(clip)
SelectEvery(clip,2,1)      # идентично SelectOdd(clip)
SelectEvery(clip,10,3,6,7) # выбирает кадры 3, 6, 7, 13, 16, 17, ... из клипа источника
SelectEvery(clip,9,0)      # выбирает кадры 0, 9, 18, ... из клипа источника
