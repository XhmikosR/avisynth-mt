SuperEQ

SuperEQ (clip, string filename)

Shibata Super Equalizer является графическим параметрическим эквалайзером (регулятором громкости разных частот) - плагином для winamp. Этот плагин использует  16383-кратный FIR-фильтр на основе алгоритма быстрого преобразования Фурье (БПФ = FFT). Эта регулировка  очень точная. Регулировка может задаваться независимо для каждого канала. SuperEQ в оригинале написан Naoki Shibata.

Фильтру SuperEQ требуется файл настроек эквалайзера  foobar2k. Сам эквалайзер можно найти на вкладке DSPManager указанного сайта, настройки регулируются и сохраняются с его помощью.

Некоторые готовые установки можно загрузить отсюда.

Этот плагин оптимизирован для процессоров, которые имеют кеш не менее  128 кБ (16383*2*sizeof(float) = 128k). Этот плагин  не может эффективно работать с процессорами серии K6 (покупайте Athlon!!!).

Параметры:

filename Файл предустановок foobar2k-эквалайзера.

Пример:

Использовать файл предустановок "Loudness" (Тонкомпенсация) для эквалайзера таким образом:

SuperEq("C:\Equalizer Presets\Loudness.feq")
