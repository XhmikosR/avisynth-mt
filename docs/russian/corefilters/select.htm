SelectEven / SelectOdd

SelectEven (clip)
SelectOdd (clip)

SelectEven формирует выходной видео-поток, используя из входного клипа только кадры с чётными номерами. SelectOdd является его нечётным аналогом.

Так как кадры (в AviSynth) нумеруются с нуля, SelectEven реально выбирает первый, третий, пятый, ... кадры по человеческому счётному соглашению.
