требования:
  • цветовое пространство RGB

лицензия: открытые исходники

Описание

Воспроизводит использование amp-файлов Photoshop и файлов цветовых кривых Gimp.

Синтаксис

GiCoCu (clip, string filename, bool "alpha", bool "photoshop")

Параметры

- опциональная поддержка альфа-канала:
если alpha = true и цветовой формат = rgb32, то альфа канал видео также обрабатываеися
по умолчанию: false

- поддержка файлов cur/amp (формат Photoshop и плагина от bugsbunny для VitrualDub)
если photoshop = true то плагин ожидает Photoshop amp-файл вместо Gimp-файла кривой цвета
по умолчанию: false
