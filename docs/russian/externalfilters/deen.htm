Deen

Общая информация

Автор: MarcFD
Версия: Beta 2
Загрузка: http://ziquash.chez.tiscali.fr/
Категория: пространственные сглаживатели
Требования:  
  • YV12 цветовой формат (Colorspace)
  • поддержка ISSE

Описание

deen является набором ассемблерно-оптимизированных шумоподавителей.

Использование

deen (clip, string "name", int "mode", int "thrY", int "thrUV", float "ti")

name :
здесь выбирается метод шумоподавления (c3d, a3d, a2d, m2d).
По умолчанию: "c3d"

mode (режим):
 с методом "c3d" параметр определяет весовую матрицу.
- 0 <121> матрица (по умолчению)
- 1 <111> матрица (более мягкая)
с другими методами - это пространственный радиус.
По умолчанию: 0

thrY, thrUV :
пороги значений яркости (luma, Y) и цвета (chroma, UV).
Большие значения дают большее смешивание.
По умолчанию: 10,12

ti :
временнОе воздействие (temporal influence).
максимальное значение означает локальные временные различия (local temporal difference)
позволяет разрешить 3d-фильтрацию.
По умолчанию: 3

Методы (methods)

Несколько методов реализовано:

- данная версия является предварительным показом режима "c3d".
Он полностью функционален и его результат
очень близок к результату фильтрования Convolution3D от vlad.

- попробуйте также режимы "a3d" и "a2d". Это мои собственные реализации 2d/3d алгоритмов шумоподавления. Они на стадии бета.
Стоит попробовать deen("a3d",1,10,12) как альтернативу c3d с матрицей <111>
или deen("a2d",2,10,12) для 5x5 пространственной фильтрации.

- "m2d" на альфа-стадии.
Но все-же стоит глянуть на deen("m2d",1,10,12)
(mmx-оптимизированный вариант реализации 2d msoften2 / 21x21)
