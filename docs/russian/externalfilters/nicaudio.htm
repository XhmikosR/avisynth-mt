Описание

Аудио плагины для MPEG Audio/AC3/DTS/LPCM. NicLPCMSource ожидает сырые (несжатые, raw) LPCM или LPCM WAV файлы. Однако, в настоящее время он поддерживает LPCM WAV файлы только 2-канальные.

Синтаксис

AC3: NicAC3Source ("filename.ac3", int "downmix")
DTS: NicDTSSource ("filename.dts", int "downmix")
MPA: NicMPASource ("filename.mpa", int "downmix")
LPCM: NicLPCMSource ("filename.lpcm", int "sampleRate", int "sampleBits", int "channels")

Параметры

Примечание: downmix указывает максимальное число каналов на выходе. Это единственный опциональный (необязательный) параметр.

Пример:

LoadPlugin("NicAudio.dll")
NicAC3Source("c:\File.AC3")

или

NicLPCMSource("c:\File.lpcm", 48000, 16, 2)

Фильтры скомпилировал и собрал вместе Nic. Все под лицензией GPL.
