ColorBars

ColorBars (int "width", int "height", string "pixel_type")

Le filtre ColorBars produit un clip contenant les barres de couleurs SMPTE ramenées à une taille d'image donnée. Le clip renvoyé à les propriétés suivantes: 640x480, RGB32 [16,235], 29.97 fps, 1 heure de durée.

Un ton de test est également généré. Ce ton de test est une sinusoïde de fréquence 440Hz à la fréquence d'échantillonnage 48KHz, 16 bit, stereo. Ce ton est produit dans le haut parleur DROIT. Il est activé et déactivé toute les secondes.

Dans v2.56, pixel_type= "YUY2" ou "YV12" a été ajouté (par défaut "RGB32"). Cela est pratiquement équivalent à: 

ColorBars().ConvertToYUY2(matrix="PC.601") # Ne restreint pas l'échelle de la luminance

Quand un clip YUV est généré, les changements de couleurs sont arrangés pour se produire dans l'alignement d'une bordure de chrominance.

Note : Les versions YUV contiennent des valeur de pixels inférieures à 16 (InfraBlack/InfraNoir)

Changements:

v2.56 ajout de pixel_type="YUY2", "YV12"
