Blur / Sharpen

Blur (clip, float amount)
Blur (clip, float amountH, float amountV)
Sharpen (clip, float amount)
Sharpen (clip, float amountH, float amountV)

C'est un simple filtre de flou utilisant un kernel 3x3. L'argument le plus grand que l'on peut utiliser pour Blur est à peu près 1.58, ce qui correspond à un flou effectué avec échantillons de 3x3 pixels. Une valeur de 1.0 vous donnera un kernel (1/4,1/2,1/4). Si vous souhaitez appliquer un flou de type Gaussien sur un rayon plus important, je vous conseille d'utiliser plusieurs instances du filtres Blur(1.0) à la suite (ça ne vous rappelle pas le triangle de Pascal?) 

Des arguments négatifs fournis à Blur augmente la netteté de l'image. En fait, Sharpen(n) est seulement un alias de Blur(-n). La plus petite valeur utilisable avec Blur est -1.0; la plus grande valeur avec Sharpen est 1.0. 

Vous pouvez spécifier 2 arguments pour rendre indépendantes les valeurs horizontales et verticales. De cette manière, vous pouvez utiliser Blur(0,1) pour filtrer seulement verticalement; par exemple pour mélanger des lignes entrelacées ensembles. Par défaut amountV=amountH.
