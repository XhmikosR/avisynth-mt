FreezeFrame

FreezeFrame (clip, int first_frame, int last_frame, int source_frame)

Le filtre FreezeFrame remplace les images entre first_frame et last_frame avec une copie de source_frame. La piste audio n'est pas modifiée. Ce filtre est utile pour remplacer une image avec un défaut par une image ne contenant par de défaut.
