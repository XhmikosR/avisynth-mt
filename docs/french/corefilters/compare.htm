Compare

Compare (clip_filtered, clip_original, string "channels", string "logfile", bool "show_graph")

Compare renvoie le clip filtré (premier argument) avec du texte (Mean Absolute Difference (moyenne absolue des différences), Mean Difference (moyenne des différences) & PSNR) ainsi que le graphe de PSNR de MarcFD (si l'argument show_graph est true (vrai)). A partir de v2.53 , 'Overal PSNR' est également affiché à l'écran mais n'est pas présent dans le fichier log. La chaîne de l'argument channels est une combinaison de R,G,B [,A] ou Y,U,V, en fonction des clips sources. Une chaîne vide (par défaut) signifie soit "YUV" ou "RGB". Si logfile est non nul, le résultat sera écrit dans un fichier du nom indiqué et ne sera pas affiché sur le clip. C'est plus rapide et plus pratique si vous devez comparer un nombre important d'images.

  

# Affiche les différences sur l'image
Compare(clip1, clip2)
# Pour créer un fichier log:
Compare(clip1, clip2, "", "compare.log")
# Compare seulement les canaux de chrominance de 2 clips YUY2.
Compare(clip1, clip2, "UV")
