MonoToStereo

MonoToStereo (left_channel_clip, right_channel_clip)

Ce filtre convertit deux signaux mono en un signal stéréo. Cela peut être utile quand vous avez modifié séparément chacune des pistes et que vous devez les réunir.

Les deux clips doivent avoir le même taux d'échantillonnage (utilisez ResampleAudio si vous avez un problème à ce niveau là). Si chacun des clips est en stéréo, le signal sera pris dans le canal correspondant de chaque clip (le canal gauche pour left_channel_clip, et vice versa pour le canal droit).
Avant le mélange des deux flux, les échantillons sont convertit dans le même type d'échantillons que left_channel_clip.

Dans v2.5, cette fonction est définie par MergeChannels.

  

#Réunit deux signaux wav séparés pour former un signal stéréo:
left_channel = WavSource("c:\left_channel.wav")
right_channel = WavSource("c:\right_channel.wav")
return MonoToStereo(left_channel, right_channel)
