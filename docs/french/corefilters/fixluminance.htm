FixLuminance

FixLuminance (clip, int intercept, int slope)

Mon magnétoscope à la mauvaise habitude de donner une image plus claire en haut qu'en bas. Le but de ce filtre est de progressivement assombrir le dessus de l'image pour compenser cela. Si vous remarquez un problème similaire avec votre magnétoscope, contactez moi et je vous donnerai une explication plus pécise de la manière dont ce filtre fonctionne. Sinon, vous pouvez faire comme s'il n'existait pas.

Ce filtre fonctionne uniquement avec une source en YUY2.
