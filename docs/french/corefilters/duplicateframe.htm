DuplicateFrame

DuplicateFrame (clip, int frame)

DuplicateFrame est le contraire de DeleteFrame. Ce filtre duplique l'image donnée en tant qu'argument. Comme pour DeleteFrame, la piste son n'est pas modifiée.
