MessageClip

MessageClip (string message, int "width", int "height", bool "shrink", int "text_color", int "halo_color", int "bg_color")

MessageClip renvoie un clip qui affiche du texte; ce filtre est utilisé en interne par AviSynth pour afficher des messages d'erreurs
La police Arial est utilisée avec une taille comprise entre 24 points et 9 points. La taille est choisie pour s'accorder si possible avec la largeur (width) et la hauteur (height) définis dans les paramètres.
Si shrink (réduction) est true (vrai), la résolution du clip est diminuée, si nécessaire, pour s'ajuster au texte.
