Info (clip)

Présent depuis v2.5. Ce filtre affiche des informations sur le clip étudié dans son coin supérieur gauche. Les informations consistent en l'espace de couleur, la taille de l'image, le fps (nombre d'image par secondes), indique si le clip est basé sur des champs(si vous avez appliqué le filtre SeparateFields) ou sur des images (si vous n'avez pas appliqué le filtre SeparateFields), indique à AviSynth si le clip est de type champs inférieur (le cas par défaut en utilisant AviSource) ou champs supérieur en premier, indique si un flux audio est présent, le nombre de canaux, le type des échantillons, et le taux d'échantillonnage. Dans v2.55 un drapeau CPU à été ajouté indiquant les optimisation supportées.

Exemple:

AviSource("C:\filename.avi").Info

Renvoie une video avec les infos suivantes indiquées dans le coin supérieur gauche: 

Frame: 12 
ColorSpace: YV12 
Width: 720 pixels, Height: 432 pixels 
Frames per second: 25.0000 
Fieldbased (Separated) Video: NO 
Parity: Assuming Bottom Field First 
Video Pitch: 720 bytes. 
Has Audio: yes 
Audio Channels: 6 
Sample Type: integer 16 bit 
Samples Per Second: 44100
CPU detected: x87 MMX ISSE SSE 3DNOW 3DNOW_EXT

Changements:

v2.50 Version initiale
v2.55 Ajout des optimisations CPU supportées
